Maqaa Abbaa, Ilmaafi Afuura Qulqulluu Waaqayyo tokko Ameen!

Waaqayyo tokko kan ta’u maqaaa Abbaa, Ilmaaf Afuura qulqulluu Waaqayyo tokkootti amanaa dirsaana kadhannaa, qulqulluu Afiniin kana naan kadhadha. Akkanas jechaa, Alifaa’eel, laa’el Gi’eel
maqooliin dhoksaa waaqummaa kana waaqayyo qulqulluu Afiniinitti, Asibuuriin diinota isaa irraa akka fayyisuufi akka gargaaruuf yeroo erge, kan itti himeef kan kenneefidha.

Yaa Gooftaa Asibuuriin akka gargaarte, akkasuma ana tajaajilaa kee ……………… dhipina fooniifi dhiphina lubbuu irraa na baraari. Yeroo lolaafi yeroo waraanaa harka diinaatti booji’amee kufuurraa na fayyisi Ameen!

Yaa ergamaa fayyaa Afiniin, nagaan siin jedha. Dhugomtota bosonaa keessatti kan tiksitu yaa Afiniin, nagaan siin jedha. Dhiiga isaaniin kan goyyoman, kan cuubaman, wareegamtootaafi jabina ofi humna kan taatuufi yaa Afiniin nagaan siin jedha. Kan dhiphatan dadhabina irraa aara kan galchisiistu yaa ergamaa nagaan siin jedha.

Yeroo bokkaan na rukkutu, jaanXilaa warqeefi daabee ifaa kan taatu yaa Afiniin nagaan siin jedha.

Yaa Afiniin Hoggaanaa ergamootaa, jeequmsa kan ummu, dubbii haadhaa’aa, ergamaa du’aa baayyee sodaachisaa kan ta’u irraa na tiksi, na fayyisis, bara baraan Ameen!

Yaa Abbaa Keenyaa……………….
