Maqaa Abbaa, Ilmaa Afuura Qulqulluu Waaqayyo tokkoo Ameen!

Waaqayyo tokko kan ta’u, Maqaa Abbaa, Ilmaaf Afuura qulqulluu, dirsaane ergamaa Faanu’eel kana nan kadadha akkanas jechaa.
Qirotoyaal, Dertiyaal, Qitiyaal, Aawusiifiyaal Maqoolii dhoksaa Waaqummaan kun, qulqulluu Faanu’eel dursee dhala namaa fuula Waaqayyo duratti kan maahii baasan uumaa wajjin kan lolchisiisan lolchisiisan lolituu diyaabiloos, leegiwoonota fuula Waaqayyo duraa haxaa’ee fuudhee laga yordaanosiif bosona qiisoonitti kan ittiin ittisudha.

Harras akkasuma, ana tajaajilaa kee ……………… diina isaa biyya lafaa irraa akka balleessuuf jecha, seexana kan hari’atu qulqulluu Faanu’eeliin gara kootti akka ergiituuf fedha kee haa ta’u Ameen!

Yaa laboobaa Faanu’eel nagaan siin jedha. Yaa adamsaa ulfaataa fannoo nagaan siin jedha. Dhala namaa kan ari’atu bineensa didaa kan ajjeestu waraanaa afuuraa yaa Faanu’eel nagaan siin jedha. Yaa uffannaa galata kan hagugifamtee nagaan siin jedha. Iyyeessa badhadheessuuf carraan kan siif kenname yaa Faanu’eel nagaan siif haa ta’u. Dhiphina adda adda kan salphistu yaa ergamaa nagaan siin jedha.

Hogganaa ergamoota yaa Faanu’eel, jeequmsa kan uumu, haafaan hadha’afi hamaa ergamaa du’aa irraa, kochoo kee lamaanu irraa qaama koo gaadeddeesite yeroo hundaafi sa’a hundaa akka na eegduufi na fayyistuuf jecha nagaan siin jedha. Badii adda addaafi cubbuudhaan. dhala namaa Waaqayyoo wajjin kan wal lolchisiisan bakka mootummaa samii Waaqayyoo irraa seexanoota kan ariitu yaa Faanu’eel nagaan< siin jedha.

Hammeenya adda addaatiin, dhala namaa Waaqayyo wajjin yeroo wal lolchisiisan fuula Waaqayyoo duraa seexana kan ari’atee yaa Faanu’el nagaan siin jedha. Ati ergamaa difamaa tihoo!

Dhala namaa fayyisuuf tattaafataa kan taate yaa Faanu’eel nagaan siif haa ta’u. Seexanoota narra ittisi ani sadarkaa qaamaan kanan jirudhahoo. Yaa Faanu’eel nagaan siif haa ta’uhoo, diinni koo dhokatee, eebboon cinaacha koo akka hin waraanneef, ana irratti humna fayyisummaa keetii ibsi. Yaa Faanu’eel, gargaarsummaa keefi humna fayyisummaa keetii yeroo ibsitu hunda yeroo bari’uufi dhi’u kabajaan nagaan siin jedha.

Ergamaa Raamaa yaa Faanu’eel nagaan siif haa ta’u hoo, jireenya sabummaa irratti rakkina uumuun diina koo kan ta’u diyaabiloosii irratti Goraadee kee ta’u diyaabiloosii irratti Goraadee kee luqiisi, taayitaa lubummaa samiifi lafaa kan siif kenname yaa Ergamaa Faanu’eel nagaan siif haa ta’u jechaa, si’a torbaatamii torbaatama sobaan badii raawadheera hoo, haa bulu haa oolu otoo hin jedhin hidhaa cubbuu irraa jabinaan na hiiki.

Tattaafataaf Ergamaa nagaa yaa Faanu’eel nagaan siif haa ta’u hoo, yeroo hundumaa na tiiksi. Keessumattuu guyyaa har’aa kana na fayyisi, gargaarsa jabaan dhadhabichi ni fayyahoo.

Yaa Gooftaa qondaala ergamaa Faanu’eel Iyyesuus Kiristoos. Loltuu Diyaabiloosi irraa, dogoggorsiisaa gidiraa seexanaa irraa, saamaafi tolcheeettuu irraa ana tajaajilaa kee …………….. hanga bara baraatti tiksi na fayyisis ameen!
