Maqaa Abbaa Ilmaa, Afuuraa qulqulluu Waaqayyo tokkoon Ameen!

Waaqayyoo tokko kan ta’u, Maqaa Abbaa Ilmaa, Afuura Qulqulluun naasuufi sodaa adda addaa irraa akka fayyuufi dirsanaa qulqulluu Gabri’eel kana akkana jechaa nan kadhadha. Madinani’el, Hiyaa wutaa’el, awukaa’el, terbutaa’el, gi’eel, zi’eel, bu’eel, Maqoolii Waaqayyoo kana, Waaqayyo qulqulluu Gabri’eeliif kan kenneef. Ergamaan diyaabiloosii wajjiin wal morkii yeroo jalqabu, akka ittiin injifatuufi akkasumas giiftii keenya qulqulletti Maariyaamiin misiraachessuf yeroo ergame, jecha isa irraan kan ka’e akka isheen hin rifanneefi, hin naaneef humna akka ta’uufiifidha.

Yaa Gooftaa, ana tajaajilaa kees akkasuma humna jabaa kee naa kenniitii naasuu halkaniif guyyaa irraa naafayyisi. Ameen.

Samii gubbaa irraa gara dubrootti kan ergamte yaa Gabri’el nagaan siin jedha. Misiraachoo kan himtu yaa ergama nagaan siin jedha. Jechi waaqayyo dubroo Maariyaam irra akka dhalatu jecha barsiistee kan amansiistu yaa ergama nagaan siin jedha. Barbadaa samii yaa Gugee nagaan siin jedha. Gammachuu jaba kan dubbatu yaa ergama nagaan siin jedha. Harka du’aa irraa, kan fayyistu yaa ergamaa nagaan siin jedha. Araarsummaa kee halkaniif guyyaa abdii waan taasifannuuf tiksummaan kee nuraa adda hin ba’in. Lubbuufi fooniin nu gargaari, jaallee Waaqayyootiihoo, bara baraan Ameen!

Yaa Abbaa Keenyaa…………….
