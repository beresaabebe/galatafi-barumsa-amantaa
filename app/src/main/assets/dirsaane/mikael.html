Maqaa Abbaa Ilmaa Afuura Qulqulluu Waaqayyoo tokkoon ameen!
Maqaa Abbaa Ilmaa Afuura Qulqulluutti amanuudhaan, Ogummaa (Maqoolii) Waaqayyo wajjin walii galchisiisuun kan kadhatamu dirsaana (seenaan) qulqulluu Miikaa’eel Kanaadha.
Waaqayyo, waa’ee gadi ofi qabummaa isaaf waa’ee amanatamummaa isaaf, waa’ee kabaja isaaf, Qulqulluu Miikaa’eeliif kan kenne Maqooliin Waaqummaan kun guddaf olaanaa, ulfaataafi mo’aadhas.
Dursee, Samii irratti waraanni yeroo ta’e, qulqulluu Miikaa’eel Marraticha waraanee gara lafaatti kan inni ittiin kuffise, Maqaa Waaqummaa kanaanidha. Yaa Gooftaa anis diinota koo warra hin mullanneef warra mul’atan akkan injifadhuuf jecha qaama gadi ofiqabumsaaf ta’ee jechoota akkana jedhee nan kadhadha.
Eekkos, piikaagiiloo, Aamee’I,Eeke, Eenkaa’I kaazee’I Arinaa kee. Keem kaayoon, piikagiiloo, Aasi’aalooti, Assisoomu, Aari’eem. Yaabeesitiga, paa’Irgaa. Yabisiit fee’e Eelikaa’I Iinikaa’i, bi’ilaakiif Aari’haaloo Aawootsi. Kabajamoo kan ta’an jechoota kanaafi jabaa Maqootiin Waaqayyoo harkaa Waaqayyoo irratti kan argaman qooqa Ibiraayisxiitiin jechoota hundaa ol akka ta’an kan beekamaniif kan hubatamanidha.
Namni kitaaba kana barreessus, Maqaa dhooksaa jabaa kana kitabicha irra hin basin, baka qulqulluu kaa’ee kabajaan haa eegu malee. Diina isaa hanga bara baraatti mo’ee isaaf dahoo ni taa’aafihoo.
Halkaniif guyyaa, badiif kan asiif achi dadeebi’an, lolitoonni diina keenya diyaabiloos, Mo’aa an ta’ee dheekkamissa Waaqayyoodhaan gonkumaa haa barbada’aan. Ameen.
Kadhannaa qulqulluu Miikaa’eeliif kadhannaa qulqulluu Gabri’eel, Kadhannaa qomoo ergamootaa sagalitamii salgan, Kadhannaa Luboota Ergamoota samii diigdamii afur, kadhannaa beelladoota arifaniin, afuurri xuraa’aafi diinni haa barbadaa’an. Ameen.
Jecha Ilma Waaqayyo, Waangeela kan barreessan Maatiwoosiifi Maariqos, jecha Luqaasiifi Yohaannisiin. Taayitaa Pheexiroosiifi Phaawuloosiin kan balaleeffaman haa ta’an.
Lubbuufi fooniin, keessa afaaniin qulqulleettii kan taate dubroota hundaa irraa adda baatee ulfaattuufi kabajamtuu kan taatu, abdii uumama hundaa kan taate Gooftaa kan deesse, humna kadhannaa Dubroo Maariyaamiin, gargaartoota seexanaa, daba kan raawwatan tolfattoonni attatamaan haa ittifaman. Ameen.
Halkaniifi barii, guyyaadhaafi waarree booda ganamaa hanga galgalaatu, galgalaa hangaa ganamaatti, qaama nama kan rifaachisiisan, lollituu Diyaabiloos, seexanoota hundaa, dachee lafaa kan jabeessan, torban humnaa qilleensootaatin akka aaraa aaranii, akka duumeessaa bittinaa’anii haa badan. Ameen.
Angafa Ergamoota hundumaa kan taate yaa qulqulluu Miikaa’eel, Waaqayyo bira kan siif keenname, humna kadhannaa Waadaatiin, kan na mormaniif kan na morkatan diinota koo mullataniif hin mul’anne irraa halkaniif guyyaa ana tajaajilaa kee hanga bara baraatti na eegi. Ameen!

Maqaa Abbaa Ilmaa Kan Afuura Qulqulluu Waaqayyo tokkoon Ameen!

Waaqayyoo tokko kan ta’e Maqaa Abbaa Ilmaa, Afuura Qulqulluutti amanee, kiiyyoo seexanaa irraa miliquuf lola seexanaa irraa fayyuuf kadhannaa Waaqummaa kana akkana jechaa nan kadhadha.
Eekkoos, Asillee, Eephaa, Eenkami kaam, Eebookaa, Kiiroom, Lekaaf, fee’ee, Lookee, We’aarinaakee, Guaa’I weabiso olaa, asibeek, abesku’i amee’i, awotsi’I, eeqe’i awootsi, kaabi’ireewu, arihaaLaay oon, Yaabisaat. Tse, Tse,Tse,Tse,Tse,Tse,Tse. Yaa Gooftaaa, dhokataafi guddaa kan ta’an maqootii kee kanaafi ajajaa Ergamootaa kan ta’u, kadhannaa qulqulluu Miikaa’eeliin lammaffaa gammachuu kan misirachoo, araarisummaa qulqulluu Gabri’eeliin, sadaffaa dhiifama kan kadhattu deessuu kee qulqulleettii Dubroo Maariyaamiin ana tajaajilaa kee dhiphina fooniif dhiphina lubbuu irraa na dhoksi. Ameen!
Yaa ergamaa Miikaa’eel nagaan siin jedha. Yaa ergamaa iyyeessa kununisituu nagaan sin jedha. Warra rakkatan garigaaruuf kan ergamtu yaa Ergamaa nagaan siin jedha. Amanatamuudhaan kan si kadhatu hundaafi eebba kochoo keetti kan gaaddesistuuf yaa Miikaa’eel nagaan siin jedha. Kufuuf kan gitintiruu akka hin kufneef daftee kan hordoftu yaa ergama nagaan siin jedha.
Waaqayyo Gooftaan keenyaa harka isaa, tolummaan guutameen akka nu eebbisuuf, fuula Uumaa duratti, sagaddee dhiifama kan kennisiistu yaa Miikaa’eel nagaan siin jedha.
Yaa Waaqayyoo, ergamaa qulqulluu Miikaa’eel ana tajaajilaa kee……………….dheekkamsa qophaa’eefi dheekkamsaa ajajamee irraa na dhoksi bara baraan Ameen!
