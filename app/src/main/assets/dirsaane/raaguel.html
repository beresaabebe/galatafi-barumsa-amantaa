Maqaa Abbaa Ilmaa Afuura Qulqulluu Waaqayyo tokkoon Ameen!

Waaqayyoo tokko kan ta’u maqaa Abbaa, Ilmaafi Afuura Qulqulluutti amane, Waaqayyo tolummaa isaatiin akka na daawwatuufi qulqulluu Raagu’eelis eebba isaatiin akka narraa addaa hin baaneef, dirsaane qulqulluu Raagu’eel kana nan kadhadha.
Akkanas jechaa Akkaa’I, Beeqaa’I kisbi’eel, Amaanu’el, Aahidi’eel, Bu’eel, Giti’eel, Aahaliyaal, Ta’aagaasiyaal, yaawuhiyaal, yaawuyaal, musaaxiyaal, Aawusiifiyaal, Aawukitiyaal, mardiyaal. Waaqayyo jechoota kana, Sammuu qulqulluu Raagu’eel keessatti kan inni qopheesse, Mootii Novaamaaniin akka gargaaruufi yeroo eergedha.

Ani tajaajilaa isa ……………………. gargaarsi isaafi tolummaan isaa akka narraa adda hin baaneef nan waawwadha, nan kadhadha. Culullee samii, simbirroo Raamaa kan jedhamtu yaa Raagu’eel nagaan siin jedha. Simbo surraan kan aggugamte yaa Raagu’eel nagaan siin jedha. Waaqayyoon tattaafannaan kan galateeffatu, Yaa Raagu’eel nagaan siin jedha. Afuuraan kan dhiphataniif yaadaan kan dadhabaniif haara galifii sammuu isaanii, kan taatu yaa Raagu’eel nagaan siin jedha.

Gargaaraa Moototaa Noovaamaa kan taate yaa Raagu’eel nagaan siin jedha. Rakkina qulqulleettii Aarseemaafi kan geesse yaa ergamaa Raagu’eel nagaan siin jedha. Badhasaa warqee galata kan badhaafamte yaa Raagu’eel nagaan siin jedha. Yaa ifa fuula Gooftaan nagaan siin jedha. Jechi sagalee keetii, dammaafi aanan caalaa kan mi’aa’uu yaa Raagu’eel naggaan siin jedha. Yaa qulqulluu Raagu’eel afaan hadhaa’aa ta’een, jeequmsa kan uumu, diinaafi seexana kan ta’u ergamaa du’aa irraa, humna kadhannaa keetiin qaama koo, hojjaa isheefi dalga ishee eegi. Ameen!

Raagu’eeliif Waaqayyo kan taatu yaa Waaqa Raagu’eel, cubbuu koo narraa haqii, rakkina fooniifi rakkina lubbuu irraa na baraari hanga bara baraatti Ameen!
