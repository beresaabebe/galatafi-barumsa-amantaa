Maqaa Abbaa Ilmaa, Afuura Qulqulluu Waaqa tokkoon Ameen!

Waaqayyo tokko kan ta’u maqaa Abbaa Ilmaa Afuura Qulqulluutti amanee dirsaana qulqulluu Rufaa’eel kana, akkana jedhee nan kadhadha.
Birihana’el Bemebirihi’eel, Kistinaa’eel, Behidinaa’eel, Aatsidinaa’eel kunis, Qulqulluu Ergamaa Rufaa’eelis, ija Xoobit kan ittiin ibseefi intala Raaguu’eel irra bulee kan ture seexana kan ittiin irraa baase, Waaqayyo kan isaa keenneef, ifa kan keennuufi diina kan haari’u maqoolii dhooksaa Waaqayyootti.

Yaa Uumaa Gooftaa koo, armaan olitti kan jiruun kadhannaa ani kadadheen, tolummaa kee beekuamuun ana tajaajilaa kee ………………. diinota koo hunda balleessii, laphee kpp keessatti ifa fakkii naa taasisi Ameen.

Galmee mana Gooftummaa Waaqayyootti ajajaa kan taatu, yaa Rufaa’eel nagaan sin jedha. Qurxummii isa guddaa fannoo Gooftaan ifattee bakka jirutti akka dhaabaattu kan taasiste yaa Ergamaa nagaan siin jedha. Uumaa kee fuul duratti galatni kee kan miidhageefi kan fudhatamee yaa Ergamaa nagaan siin jedha. Waaqayyo waaqotaa kan ta’uu Waaqayyoon kan galateeffatu yaa ergamaa nagaan siin jedha.
Afaan hadhaa’an jeequmsaa kan uumu diina diyaabiloosiif jabaa kan ta’u ergamaa du’aa irraa ganamaafi galgala, halkaniif guyyaa akka na tiksituuf yaa Rufaa’eel nagaan siin jedha.

Gooftaa qulqulluu Rufaa’eel taatu yaa Waaqayyo, maqaa waaqummaa kee kanaan dhiphina fooniifi dhiphhina lubbuu irraa na fayyisi hanga bara baraatti. Ameen!
