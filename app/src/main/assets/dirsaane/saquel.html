Maqaa Abbaa, Ilmaa Afuura Qulqulluu Waaqayyoo tokkoon Ameen!

Waaqayyo tokko kan ta’u, maqaa Abbaa, Ilmaafi Afuura Qulqulluutti amanee, dirsaane ergamaa qulqulluu Saaqu’eelii akkana jedhee nan kadhadha.

Adidi’el, Sahili’eel, Bu’eel kan jedhamu maqoolii dhoksaa Waaqummaa kana Waaqayyo qulqulluu Saaqu’eeliif kennee mootota Hindeekee diinota isaani irraa kan inni ittiin fayyise, humna Waaqummaati.

Yaa Gooftaa ana tajaajilaa kee………………….. akka isaanii maqaa jabaafi itti hin bu’amnee kanaan na gargaari, na fayyisis ameen!

Gara laafessaafi garraamii kan taate yaa Saaqu’eel, nagaan siin jedha. Ija warra hin argine kan bantu, yaa ergama nagaan siin jedha. Cubbuun namootaa hundi, akka irraa haqamuufiif kan kadhattu yaa Saaqu’eel nagaan siin jedha.

Kan dhiphatan dhiphina isaaniirraa kan boqochiistu yaa ergamaa nagaan siin jedha. Waaqayyo waaqotaa nuffii malee kan galateeffattu yaa ergamaa Saaqu’eel, nagaan siin jedha.

Yaa Saaqu’eel nagaan siif haa ta’uhoo dubbii badiifi hadha’a irraa, ergamaa du’aa, nama rifachiisuufi nama sadaachisu irraa halkaaniif guyyaa, ganamaafi galgala ana tajaajilaa kee …………….. na tiksi, na fayyisis hangaa bara baraatti ameen!
