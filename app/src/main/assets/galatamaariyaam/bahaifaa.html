1. Galata Haadha Waaqayyoo nama Itiyoophiiyaa kan ta’e qulqulluun Yaareed kan barreesse. Murteessituu dhugaa haadha Waaqayyoof tajaajilliifi galanni ni ta’aaf.

  2. Yaa Qulqulleettii, ati kan eebbifamte kangalatoomte, kan kabajamte, ol ol kan jettedha. Ati baha ifaa riqaa jireenyatti. Yaa giiftii keenya Durboo Waaqa deesse, ati kan filataman keessaa kan filatamte bultoo Waaqummaati. Uummamtoota hundumaarraa kan kabajamte, yaa Maariyaam Abbaan kan si jaallate, Ilmi si keessa bule, Afuurri Qulqulluunis sirra boqote jedhamte. Ati lafarratti Ariyaam kan taate, anga’oof iddoo Ariyaam kan bakka buute sidha. Raajota, lubootaafi mootoonni akka fakkeenya keetti Tsillaatiin waadaa kan keessashee jiru qulqulleettii, kan adda ta’an irraa adda taate hojjeetan.
    Yaa qulqulleettii Ilmi kee dhifamasaa akka nuu keennuuf nuu kadhadhu.

  3. Ilmi Waaqaa sirraa dhalatee nama ta’uusaatin, sababa keetinis gara Ariyaam Mootummaa Waaqayyootti kan dhiyaanne taane, maaqa keefi maaqa Ilma keef kan dhiyaanne taane. Abboommiin kurnan qubasaatiin kan barreeffaman, Gooftaan hundumaa, Ilmi Waaqayyoo garaa kee keessa bule. Angoosaan si jabeesse. Afuurri qulqulluunis sirra bule, si qulqulleesse, humni Waaqayyoos si jabeesse. Iyyesuusis foon kee uffate, sagalee guddaadhaanis, “Kitaabni bishaan jireenyaa garaasaatii ni burqa, akkuma jedhe, eenyuyyuu kan dhebote yoo jiraate gara kootti dhufee haa dhugu” jedhe.
    Yaa qulqulleettii ilmi kee dhiifama saa akka nuu kennuuf nuuf kadhadhu.

  4. Duraan dursee yawuxaadhaan nutti hime.Yawuxaa kan jedhaman abboommii kurnan quba Waaqayyoon barreeffamanidha. Ilmi kee Gooftaafi Fayyisaan keenya Iyyesuus Kiristoos ati qofa Abbaasaa akka taate, Innis Ilma kee qofa akka ta’e nuu barsiise. Akkasumas samiirratti haadha, lafarrattis abba hin qabu jennee amanna, dhufaatisaas Abbaasaafi Afuurra qulqulluu dhugaa kan beeku otoo hin jiraattiin samiirraa gadi bu’e jennee ni amanna.

    Garaa kee keessa bulee ji’a sagal baatteen. Samiwwan ni gammadan; laftis ni gammadde. Dhalachuu ilma keetti ergamaan misiraachoo hime. Ergamoonni samiis, “Galanni Waaqayyoof samiirratti haa ta’u,
    lafarrattis nageenyi namafis hanga eeyyamasaatti” jedhanii galateeffatan. Tiksitoonnis Betaliheemitti kan dhaga’aniif kan argan irraan kan ka’e dinqisiifatan. Namoonni ogummaa Ilma keef sagaduufi si tajaajiluuf urjii ilaalanii biyya fagootii dhufan.

    Baha biiftuu kaasee hangaa Betaliheemitti oofee kan isaan fide bakkalchi sun iddoo ati mucaa kee wajjin jirtuti dhaabate. Namoonni ogummaa sunis arganii guddaa gammadan, gara mucaa keetis galanii isa dura dhaabbatan, lafattis kufanii sagadannif. Saanduqa isaaniis bananii warqee, ixaanaafi qumbii harkaa fuudhii dhiyeessaniif. “kadhannaa lammiikee aarsaasaanii akka fudhattuuf cubbuusaaniis akka isaanii dhiistuuf kan dhufte Yaa Gooftaa keenyaa harka fuudhii sii dhiyeessine,
    Maqaa kee ulfaataafis ixaana sirraa arganne siif dhiiyessine jedhan.
    Yaa qulqulleettii Ilmi kee dhiifamasaa akka nuu kennuuf nuuf kadhadhu.

  5. Qulqulloota hundumaa keessaa qulqulluun sidha. Akkuma Taabota muka daanoon hin nyaannerraa tolfamtee, warqeedhaanis akkuma midhagdee, gatiinsaas guddaa kan ta’e inquu (Albuuda ulfaataa) calaqqisuun akkuma dibamee taatee mana qulqullummaa keessa jiraachuuf kan filatamte dubroon sidha. Akkana taatees mana qulqullummaa keessa jiraatte.
    Ergamoonni si samiis yeroo hundumaa nyaata kee siifidu ture. Akksiitti Ergamoonni si tajaajilaa waggaa kudha lama jiraatte. Kan ati dhugdus dhugaatii jireenyaati ture, nyaanni kees nyaata samii ture. Abbaan kee Daawit masanqoodhaan si galateeffate. Afuura qulqulluun guutamee baganaa rukuchaa raajiidhaan, “Yaa mucaa koo dhaga’i, gurra kees as-qabi fira keefi mana Abbaa kee irraanfadhu, mootichis midhagina kee jaallateera, inni Gooftaa keedha’oo, Isaafis sagadi ni sagaddaafis” Far. 45:9-10 jedhee faarfate.
    Yaa qulqulleettii Ilmi kee dhiifamasaa akka nuu kennuuf nuu kadhadhu.

  6. Gartuu Ergamoota qulqullaa’oo warra hin yakkinerraa kan ta’e fuula Waaqayyoo dura kan dhaabbatu, garee ergamoota warra gaarii kan ta’e Gabri’eel ergamaan missiraachoo sitti hime. Akkanas siin jedhe, “Kennaan kan guutamte gammadi Waaqayyoo si wajjin jira ati dubartoota hundumaa caalaa kan kabajamtedha. Bulchaa hundumaa kan ta’e Waaqayyoo biratti ulfina argatte.
    Kunoo ni ulfoofta Ilmas ni deessa maqaasaas Iyyesuus jette ni moggaafta. Innis jabaa Ilma Waaqayyoo anga’oo jedhama. Waaqayyoo Gooftaanis teessoo Daawit kan abbaasaa ni keennaaf, bara barafis lammii yaa’iqoob ni bita. Mootummaasaafis dhuma hun qabu. Afuurri Qulqulluu sirra ni bula, humni humni Waaqayyoos sirra jiraata. inni sirra dhalatus qulqulluudha.
    Yaa qulqulleetti ilmi kee dhifamasaa akka nuu kennuf nuuf kadhadhu.

  7. Galmi qulqullummaafi adda kan ta’an irraa adda kan taate sidha. Golgi ifaafi ulfinaa sidha. Yaa qulqulleettii Maariyaam Afuurri qulqulluu waan sirra buleefi humni humni Waaqayyoo waan sirra jiruuf siin qulqulluu qulqulleettii mana qulqullummaa adda taatetti si fakkeessine. Ati golga ifaadha jechuun keenyas jecha Abbaa kan ta’e fayyisaan keenya Iyyesuus Kiristoos garaa kee keessatti waan dhalateefidha. Innis samiirra kan jiru Goofta hundumaa, ergamootaafi hangafoonni ergamootaa, anga’ooni ijiisaanni, ifoonni mootonni, humnoonni, hangafoonni qondaaltoonni kan isa galateeffatanidha. Yeroo Ilma kee addunyaa kana keessatti deesseenii ijjiisaanii baay’ee kan ta’an kirubeeliifi kaattuun (koochoo) isaanii ja’a kan ta’an Suraafeel dallaa keessattis si golgan (si marsan).

    Duumessi ifaas si marsan, hangafoonni ergamootaa raayyaan ergamootaa samii si fuldura dhaabbatan. Kirubellifi Suraafeel addunyaa kanatti galata duran samiirra hin jirre galata haaraatiin si galateeffatan. Uumama hundumaa sassaabee kan bulchuufi kan nyaachisu Gooftaansaanii harka kee irra taa’ee akkuma daa’immanii yommuu harma hodhu argan.
    Ariyaamitti barbaadanii akkuma duraatti Abbaasaafi Afuura qulqulluu wajjin isa argatan. Gad of qabinsa Gooftaa isaanii yommuu arganis ijasaanii gara Ariyaamitti ol kaasani, kaattuu (koochoo) isaanii diriirsanii “Yaa Gooftaa hundumaa samiirratti galanni siif ni ta’a.” jedhanii Gooftaa isaanii galateeffatan. Lamaffaas, dallaa horii keessatti mucaa kee wajjin si arganii “Lafarratti nagaan (Jaalalliifi tokkummaan) ta’e” jedhan.

    Foon keenya isa sirraa fudhatee uffatee yommuu arganis “Akkam akkanatti nama jaallate” jedhanii sagadanif. Raajichi Daawitis, “Waaqayyos samiifi iddoo bultoosaarra taa’ee gara warra lafarra jiraatan hundumaatti ilaale, dhala namaa hundumaas arge; mootiin miidhagina kee jaallate, bara baraaf boqonnaan koo ishee kanadha, waanan ishee filadheef ishee keessa nan bula.” jedhee raajii dubateef.
    Yaa qulqullulleettii Ilmi kee dhiifamasaa akka nuu kennuuf nuuf kadhadhu.

  8. Adda kan taateefi gaarummaa kan qabdu, kan galatoomteefi kan eebbifamte kan kabajamteefi ol’aantuu kan taate Yaa Maariyaam masoobii warqee nyaanni jireenyaa kan keessa jiruun si fakkeessine. Nyaanni jireenyaa kunis, qulqulloota mirgasaan jiran wajjin isatti kan amaneefi lapheesaa qulqulleesse kan isarraa nyaatu hundumaaf jireenya kan kennu samiirraa kan bu’edha.
    Yaa qulqulleettii Ilmi kee dhiifamasaa akka nuu kennuuf nuu kadhadhu.

  9. Harki ogeessa kan ishee hin tolchine, keessasheettis ibsaa kan ibsan baattuun ibsa warqee sidha. Ifni garaa keeti dhufee si keessa bule, Waaqummaa isaatiinis addunyaa hundumaaf ibse, Dhala-namaarra dukkana fageesse. “Ani ifa addunyaati ifa kooti amanaa, yeroo ifa ta’ee jirutti deddeebi’aa” jedhee jechasaa nama fayyisuun nu fayyise.
    Yaa qulqulleettii Ilmi kee Dhiifamasaa akka nuu kennuufnuu kadhadhu.

  10. Gooftaa hundumaarraa argame Gooftaa hundumaa, ifa ifarraa argame, dhuguma Gooftaan hundumaa kan dhalte malee kan uumame miti. Jireenyaan, aangoodhaan Abbaasaa kan walqixxaate samiifi lafarra kan jiraatan hundumtuu kan isaan uumaman nu fayyisuuf jedhee kan samiirraa gad bu’e Kiristoos, qulqulleettii dubroo Maariyaam irraa nama ta’e. Dhufaatii isaatinis nuu ibse, Missiraachoo gammachuu guddaas nutti hime, Abbaasaa wajjinis nu araarse, gara daandii jireenyaattis nu deebise, jireenya bara baraas nuu kenne. Raajichi Isaayaas isatti amanuudhaan, Seerasaarra kan ka’e raajiidhan dinqisiifachaa “seerri (abboommiin) kee lafarratti ifadha.” jedhe. Jaarsa, lubaafi ajaja Waaqayyoo kan eege Zakkaariyaas qulqulluun ni dinqisiifate, dubbachuus jalqabe, “Waaqni keenya dhiifamasaatiin Ariyaamirra taa’ee nu daawwate, warra dukkanaafi gaaddidduu du’aa jala jiraachaa turaniif ifasaa isaanif ibsuuf dhalate. Cubbuu keenya nuu dhisuuf, nu fayyisuufis miilla keenya gara daandii nagaatti qajeelche.”
    Yaa qulqulleettii Ilmi kee dhiifamasaa akka nuu kennuuf nuu kadhadhu.

  11. Yaa giiftii keenya kadhannaan qulqullootaafi amantoota hundumaa ultuu (Maaxantii) isaanitiin lafaatii gara samiitti kan ittin ol baasan ultuu warqeerraa kan tolfamee harka hangafoota luboota samii keessa jiruun si fakkeessina. Akkasumas maqaa kee waamudhaan kadhannaa dhala nama hundumaa gara sadan adda ta’an gara Abbaa, Ilmaafi Afuura qulqulluutti ni dhiyeessu. Cubbuu lammiikee akka dhisistuuf Abbaa, Ilmaafi Afuura Qulqulluu biratii siif eeyyamameera. Akkasumas, dhala namaaf riqaa jireenyaa bara baraa akka taatuuf. Yaa qoricha addunyaa lammiikee amantootaaf qorichasaanii jedhamuu qabda.
    Yaa qulqulleettii Ilmikee dhiifamasaa akka nuu kennuuf nuuf kadhadhu.

  12. Ati lalistuu eebbifamtuudha, biqiltuu jireenyaafi fayyinaati. lalistuu jireenyaa Gannata keessa jirtuuf addunyaa kanatti biqiltuu jireenyaa taatee bakka buute. Firiin kees, firii jireenyaati, namoota addunyaa isa beekan biratti habaaboon urgaa gaarii qabu sirraa mul’ate kunis karbee mi’aa saliik, kan jedhaman urgooftuu uffata keerra jiru jedhee raajichi Daawit baganaadhaan raajii kan siif dubbatedha. Abbaan kee Salomonis “Yaa misirroo koo biqiltuu dallaan itti ijaaramtedha. Daandiin kee biqiltuu itti ijaarame, ati boolla chaappeeffamtedha. Urgooftuu mi’eessituu wajjin urgaa qabdaa, urgooftuu qo’i, urgooftuu naardos, urgooftuu matsiriiy kan jedhaman wajjin jirta. Urgooftuuqatsiimataat qanaanimos kan jedhaman urgooftuu baddaa wajjin qabda. Urgooftuu warra kana hundumaa wajjin urgaa hundumaa kan caalu urgooftuu Karbeefi Aliwi qabda. Yaa dhangatuu Gannataa, ati bishaan jireenyaa baddaarraa dhangala’uuf burqaadha.
    Yaa qulqulleettii Ilmi kee dhiifamasaa akka nuu kennuuf nuu kadhadhu.

  13. Otoo hin dhaabiniifi bishaan otoo hin obaasiin lalistee kan argamtee uleen Aroon Mana qulqullummaa keessa jirti ture. Lubootaafis ishee qopheessee ture. atis akkumashe qulqullinaan mana qulqullummaa keessa jiraatte. Mana qulqullummaa keessas ulfinaafi gammachuu guddaadhaan baate. Dhugumatti firii jireenya kan ta’e Gooftaafi Fayyisaa keenya Iyyesuus Kiristoos sirra argamee. Yaa qulqulleettii akkuma ergamicharraa dhageese otoo dhiira hin beekin Ilma argate. Ergamichi “Afuurri qulqullu dhufee sirra bula, humni Waaqayyoos si golga” jedhee sitti himee ture.
    Yaa qulqulleettii Ilmi kee dhiifamasaa akka nuu kennuuf nuu kadhadhu.

  14. Kennaan kan guutamteefi madda gammachuu kan taate yaa Giiftii keenyaa, kan maqaa Ilmakeetti amanne tajaajiltoota kee dhiiraafi dhalaaf nuuf kadhachuun siif ni ta’a. Ati ijisaani baay’ee kan ta’an kirubeeliifi kaattuunsaanii (Koochoon) jaha kan ta’an Suraafeel caalaa simboo jaallatamaa ni qabda. Isaanis fuulasaanii ni dhokafatu, Miillisaaniis akka fakkii fannootti. Kunis ibidda Waaqummaa Ilma keerraa ba’u irraa akka fayyaniif. Ati garuu Waaqummaaf bultoo taate. Waaqummaan Gooftaa si hin gubne. Laboba ibiddaa baatte malee. Waaqummaan Gooftaa si hin gubne. Ati lalistuu Phaaxooos ishee ibiddi otoo hin gubiin Muuseen arge fakkaatte.

    Waan hundumaan ibidda kan ta’e Gooftaan akkuma mukittii phaaxoos si hin gubne. Dhugumatti abdii kiristaanotaa kan taate sidha. Ilmi Waaqayyoo sirraa waan nama ta’eef sababa keen lafaa gara Ariyaamitti (Samiitti) kan dhiyaanne taane. Yaa Maariyaam warra kadhannaa keetti amananiif dhiifama kan keennisiiftu, ati lubbuu keenyaaf jireenya kan kadhattudha. Isatti, Abbaasaafi Afuura qulqulluutti amannee amantaa dhugaatti akka nuu cimsuuf gara Gooftaafi Fayyisaa keenyaa Iyyesuus Kiristoositti nuuf kadhadhu. Akkasumas, dhiifamasaa akka nuu kennuuf, baay’ina dhiifamasaatiin cubbuu keenya akka nuu haquuf, Yaa Maariyaam Gooftaa hundumaa kan deesse galanni siif ni ta’a. har’as yeroo hundumaas bara baraafis galannii haafi ulfinni Abbaa, Ilmaafi Afuurra qulqulluuf ni ta’a.
    Yaa qulqulleettii Ilmi kee dhiifamasaa akka nuu kennuuf nuu kadhadhu.

  15. Qulqulluu Efreemiifi dhalataa Itiyoophiyaa kan ta’e qulqulluu Yaareedirra kan bulee, Kadhannaan, ulfinnisheefi arjummaan Ilmaa kee jaallatamaashee; _____________________ wajjin bara baraaf haa jiraatu.
