Guyyaa Sanbataa Kirsitaanaatti Kan Kadhatamu Gooftaa Keenya Kan Deesse Galata Giiftii Keenya
Durboo Maariyaam
Kutaa Torbaffaa

  1. Yaa Maariyaam, dubartoota hundumaa keessaa kan eebbifamte, jaallamtuus jedhamte. Adda kan ta’an caalaatti adda kan taate, Tsillaattin seera kakuu ishee keessa jira kan jedhamtu kutaan lamaffaa sidha. Seerri sunis abboommii kurnaan quba Waaqayyon kan barreeffamanii dha. Otoo hin jijjiraamiin sirra nama kan ta’e fayyisaan keenya Iyyasuus Kiristoos maqaasaa isa tolaa ta’e dursee yawixaatti (jechoota kurnaan)quba Waaqayyoon bareeffaman nutti hime. Kakuu haaraafis araaarsaa ta’e. Dhangala’uu ulfaata dhigaasaatiin gartuusaa kan taaan amantoota qulqulleesse.
    Yaa qulquulleettii nuuf kadhadhu

  2. Muka daanoon hin nyaannerran kan tolfame Taabonni keessafi alaan warqeedhaan kan dibame jecha Waaaqayyo isa otoo hin jijjiramiin nam ta’e fakkaata. Kunis Waaqummaa qulqulluu jijjiiramni itti jirredha. Abbaa wajjin kan walqixxaatedha. Giiftii keenya Qulqulleettif ofiisaa missirachoo itti hime. Ogummaasaa adda ta’een sanyii dhiraa malee akkuma keenya nama ta’e Waaqummaasaan foon uffatee sirraa nama ta’e.
    Yaa qulqulleetti nuuf kadhadhu.

  3. Yaa Maariyaam fakkii Waaqayyoon kan fakkeeffaman kiruubeel kan si golgan manni qulqullummaa sidha. Yaa dubroo jechi jijjirama malee sirraa anma ta’e cubbuu keenya kan nuu dhiisu, gidiraa keenyas kan nurraa fageessu ta’e.
    Yaa qulqulleetti nuuf kadhadhu.

  4. Nyaanni dhokate kan si keessa jiru, Leemmati warqee sidha. Nyaanni kunnis,nyaata samiirraa bu’ee addunyaa hundumaaf jireenya kennudha.
    Yaa qulqulleetti nuuf kadhadhu.

  5. Ibsaa yeroo hundumaa Ifummaa qabu ibsituu warqee kan baatte sidha. Ibsaan kunis,ifa addunyaadha. Ifichis ifa dhugaa jalqaba hin qabnerraa kan argame ifa dhugaadha. Waaqa dhugaarraa kan argame Waaqa dhugaadha. Innis otoo hin jijjiiramiin sirra nama kan ta’edha. Dhufaati Isaatiinis dukkaana fi gaaddidduu du’aa jala kan jirruuf nuu ibse. Iccitii ogummaasaa adda ta’eenis foon keenya uffatee yaada laphee keenyaa gara karaa nagaatti nuuf deebise.
    Yaa qulqulleetti nuu kadhadhu.

  6. Eebbifamichi mana qulqullummaa keessaa baatee kan si baase barbaada ibiddummaa isa qabu kan baatte ultuun warqee sidha. Barbadaa kan jedhames cubbuu namaa kan haquu fi yakka namaf dhiifama kan kennudha. Innis sirraa nama kan ta’ee fi aarsaa jaallatamaa taasisee kan of dhiyeesse jechaa Waaqayyo dha.
    Yaa qulqulleetti nuuf kadhadhu

  7. Jecha waaqayyo kan nuu deesse Yaa Maariyaam gugee toltuu gammadi. Hidda Iseeyyiirraa kan baate habaaboon urgaa tolaa qabdu sidha.
    Yaa qulqulletti nuuf kadhadhu.

  8. Sanyii dhiira malee nama ta’ee kan nu fayyise Gooftaa keenya isa dhugaa Kiristoosin kan nuuf deesse yaa Maariyaaam ati akka ulee Aaroon ishee otoo hin dhaabamin, bishaanis otoo hin dhugiin lalistee argamte fakkaata.
    Yaa qulquulleetti nuuf kadhadhu.

  9. Yaa Maariyaam ayyaanaan kan guutamte ati qulqulloota hundumaa caalaa nuu kadhachuun siif ni ta’a. Ati angafoota Phaaphaasota ni caalta. Raajotaafi barsiisotas ni caalta. Suraafeeliifi Kirubeel caalaa simboo jaallatamaa qabda. Dhugumatti abdiin namumaa keenyaa sidha. Isa amanuudhaan amantaa siiriitiin akka nu jabeessuuf, dhiifamasaas akka nuu kennuuf, baay’ina arjummaasaatiin cubbuu keenya akka nuu dhiisuuf gara Gooftaafi fayyisaa keenyaatti nuu kadhadhu.
    yaa qulqulleettii nuuf kadhadhu.

  10. Kadhannaanshee, ulfinnisheefi arjummaan ilmashee_________________Wajjin bara baraaf haa jiraatu. Ameen
