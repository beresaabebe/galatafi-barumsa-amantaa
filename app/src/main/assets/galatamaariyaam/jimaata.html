Jimaata Kan Kadhatamu Gooftaa Keenya Kan Deesse Galata Giiftii Keenyaa Durboo Maariyaam
Kutaa Shanaffaa

  1. Yaa Maariyaam ati dubartoota keessaa kan eebbifamtedha. Firriin garaa keetis kan eebbifameedha. Mudaa tokko malee Waaqa kan deesse Yaa Durboo Maariyaam aduun dhugaa si keessaa bahe, Gooftaan nu uume nuu dhalate, arjummaa kaattuusaatiin (Koochoosaatiin) ofitti nu dhiyeesse.
    Yaa qulqulleettii nuuf kadhadhu.

  2. Waaqa kan deesse yaa Giiftii keenyaa, haadha ifaa waan taatef ulfinaan si galateeffanna. Kan eebbifamte, Yaa durboo Maariyaam ati ergamoota samii ni caalta, ijoollee Addaam hundumaas ni caalta. Uumama hundumaa ni caalta, kan si fakkaatu hin jiru ulfinna kee dubbachuu kan danda’u eenyuudha? Gooftaa Kirubeeliifi Suraafeeliirra bulee jiraatu dhufee waan garaa kee keessa buleef, Yaa Maariyaam ergamoonni si kabaju, Suraafeelis sigalateeffatu. Nama kan jaallatu inni garasaatti nu simate. Galanniifi ulfinni kan isaaf ta’u du’a keenya fudhatee jireenyasaa nuuf kenne.
    Yaa qulqulleettii nuuf kadhadhu.

  3. Gooftaa kan deesse yaa abdii qulqullootaa durboo Maariyaam, ati kan eebbifamteedha, firiin garaa keetiis kan eebbifamedha. Addunyaa kana dura kan ture inni nama ta’e, sirraa kan dhalate foon keenya uffate, Afuura Qulqulluus nuu kenne. Baay’ina arjummaasaatiin fakkaattotasaa nu taasise. Ulfanaafi kennaa kan fudhatan dubartoota hedduurra ai ni caalta. Yaa Maariyaam Waaqa kan deesse, ati magalaa xiqqoo Waaqayyo anga’oon si keessa buleeti. Gooftaa Kirubeeliifi Suraafeelirra jiraatu harka keetti baatte. Baay’ina arjummaasaatiin uumama foonii hundumaa kan jiraachisu inni harma kee hodhe. Kunis hundumaa kan fayyisu Waaqa keenyadha. Bara baraaf nu eega, akka isaaf sagadnuufi galateeffannuuf inni nu uumeera.
    Yaa qulqulleettii nuuf kadhadhu

  4. Durboon Maariyaam bultoo urgaafi burqaa bishaan jireenyaati. Firiin garaashee nama hundumaa fayyiseera’oo. Abaarsa keenyas nurraa balleesse nu gidduuttis ta’ee fannoosaatiin nuu araarse, du’aa ka’umsa adda ta’eenis lammaffaa nama gara Gannataatti deebise.
    Yaa qulqulleettii nuuf kadhadhu

  5. Qulqulleettiin durboo Maariyaam kan amanamte Waaqas kan deessedha. Dhala namaafis dhiifama kan kadhattudha. Cubbuu keenya akka nuu dhiisuuf gara kiristoos Ilma keetti nuuf kadhadhu.
    Yaa qulqulleettii nuuf kadhadhu

  6. Mana qulqullummaa keessatti Durboo Maariyaam dhageessistee, “Yaa durboo Maariyaam nagaan siif haa ta’u jedhee ergamichi kabajee kan natti himeen alatti waanan beeku akkan hin qabne Waaqayyo ni beeka” jette dubbatti ture, kan hin danda’amne dandeesse, kan hin murtoofneefi homtuu isa murteessuu kan hin dandeenye garaa keetti murteessite. Kennaan kan guutamte galanni kee ni baay’ata, ulfinni kee hangana hin qabu. Bultoo jechaa Abbaa taateetta. Gartuu Kiristaanota kan walitti qabduufi fayyisaa kan ta’an Sillaaseef sagaduu kan barsiiftuuf, golgi bal’aa utubaa ibiddaa Museen argee kan baatte sidha. Kunis dhufee garaa kee keessa kan bule Ilma Waaqayyoodha. Gooftaa samiifi lafa uumeef bultoo taate. Ji’a sagal garaatti isa baatte. Samiifi lafti kan isa hin murteessineef kan amanamtee sidha. Lafaa gara samiitti ol ba’uuf riqaa taate, Ifni kee ifa aduu ni caala. Qulqulloonni gammachuu guddaadhaan urjii kan argan sirraa kan dhalate bahi biiftuu sidha. Kunis dhiphinaafi ciniinsuudhaan akka deessuuf Hewaanitti kan murteesseedha. “Yaa Maariyaam ati garuu, ayyaanaan kan guutamte gammadi” sagalee jedhu dhaga’uudhaan dhufee kan nu fayyise Mootii hundumaa kan ta’e nuu deesse. Innis Abbaa dhiifamaafi nama kan jaallatudha. Kanaafuu, “Yaa Maariyaam ati kan eebbifamtedha, firiin garaa keetis kan eebbifameedha. Kennaan kan guutamte gammadi Waaqayyo si wajjin jira’oo.” jechaa akkuma ergamaan Gabri’eel sigalateeffate nutis si galateeffannaa.
    Yaa qulqulleettii nuuf kadhadhu
