KADHANNAA YEROO HUNDAA

Mallattoo Fannoo Gooftaa Koo Iyyesuus Kiristoosin qaama koo hundumaa yeroo sadii nan mallatteessa.

  1. Waaqa tokko kan ta’an maqaa Abba, kan Ilmaa, Kan Afuura Qulqulluutti,
     qulqulluufi ulfaatoo
    kan ta’an sadummaatti (sillaaseetti) amanuufi kadhachuun diina
    koo Seexana nan gana, haadha koo mana Kiristiyaanaa kana fuuldura dhaabadheen sigana. Kanaafis ragaan koo
    Maariyaamii dha,
    Addunyaa kanattis Addunyaa isa dhufuttis ishee dawwoo taasiseen yaa diina koo Seexanaa sin gana.
  2. Yaa Gooftaa si galateeffanna, yaa Gooftaa si Kabajna, yaa Gooftaa sif bitamna, yaa Gooftaa maqaa kee qulqulluu ni galateeffanna.
    Jilbi hundumtuu kan siif sagaduu siif sagadnaa; arrabni hundumtuu sif bitama. Waaqni Waaqotaa, Gooftaan Gooftotaa, Mootiin moototaa si’iidha.
    Uumaan fooniifi lubbuu sidha. Isinis yeroo kadhattan akkas jedha kadhadhaa jedhee qulqulluun ilmi kee akkuma nu barsiisetti si waammana.
  3. Yaa Abbaa keenya Samiirra kan jiraattu maqaan kee haa qulqullaa’u. Mootummaan kee haa dhufu jaalalli kee samiirratti
    akkuma ta’e akkasumas lafarratti haa ta’u. Kan nu ga’u soora keenya ara’aaf ga’u nuuf kenni,
    yakka keenyas nuuf dhiisi, nutis kan nu yakkaniif akkuma dhisnuuf, qorumsatti nu hin galchiin; hamaarraa nu oolchi malee, Mootummaan kan keetti’
    oo     humnis galannis bara-baraan Ameen!
  4. Nagaa Ergamaa Gabri’eel qulqulluutiin Yaa Giiftii keenya Maariyaam nagaa siin jenna. Yaada keetiin durboodha, foon keetiinis durboodha.
    Yaa hadha Waaqayyoo isa mo’aa siif galanni haa ta’u. Ati dubartoota keessaa kan eebbifamteedha firiin gadameessa keetis kan eebbifameedha. Gammadi yaa ayaana qabeetti
    Waaqayyoo si wajjin jira’oo     Gooftaa keenya Iyyesuus Kiristoos irraa dhiifama nuuf kadhadhu, cubbuu keenya akka nuuf dhiisuuf. Hangaa bara-baraatti Ameen!

HUNDEE AMANTAA

  1. Hundumaa kan uume Waaqa tokko kan ta’u Waaqayyo Abbaatti ni amanna. Samiifi lafa kan uume, addunyaa mullatu kanaafi kan hin mul’anne osoo hin uumamin, isa wajjin kan ture
    Ilma Abbaa kan ta’u Waaqa tokko Iyyesuus Kiristositti ni amanna; Ifa Ifarraa argame, Waaqa Waaqa dhugaarraa argame, kan dhalate malee kan hin uumamne, amalummaa isaatin Abbaa wajjin kan walqixxaatu, hundumtuu kan isaan
    ta’e, samiirras lafarras kan jiru isaatiin malee homtuu kan ta’e hin jiru. Nuu fayyisuuf samiirraa gad bu’e, hojii Afuura Qulqulluutiin Qulqulleetti dubroo Maariyaamirraa nama ta’e.
    Nuuf jedhee fannifame, bara Philaaxoos Pheenxichaattis rakkina arge, du’e awwaalame, guyyaa sadaffaattis du’a mo’ee ka’e; akkuma ktaabolii qulqullaa’ootti barreeffame, ulfinaafi galataan gara samiitti ol ba’e,
    mirga Abbaasaa taa’e; lammaffaas warra du’aniifi jiraattota irratti murteessuf galataan ni dhufa. Mootummaansaas dhuma hin qabu. Afuura qulqulluutti ni amannaa, Innis Gooftaa jireenya nuuf kennudha, Abbaarraa kan ba’e, Abbaafi Ilma wajjin tokkummaatti ni sagadnaaf, ni galateeffannaan.
    Innis, Raajotarra bulee kan dubateedha. Hundumaan ol kan taate Duuka bu’oonni kan walitti qaban(hundessaan) ulfaattuu mana Kiristaanaa tokkittiitti ni amanna. Cubbuun kan ittiin haqamu Cuuphaa tokkittiitis ni amanna; Warri du’aan akka ka’aan ni abdannaa, jireenya isa dhufuf hangaa bara-baraatti Ameen.
  2. Yaa Waaqayyo Mooticha, Qulqullu, Qulqulluu Qulqulluujedhamtee ni galateeffamta. Galanni kees samiifi lafarratti kan guutedha. Yaa Kiristoos siif sagadna, arjaa Abbaa kee isa samii wajjin, fayyisaa kan ta’e afuura Qulqulluu wajjinis siif sagadna. Gara Addunyaa kanaatti dhuftee nu fayyisteetta’oo.
  3. Abbaaf, Ilmaaf, Afuura Qulqulluuf sagada tokkittii nan sagadaf (3x) tokko yommuu ta’an sadii, sadii yemmuu ta’an tokko, Qaamaan sadii yemmuu ta’an Waaqummaadhaan tokko kan ta’aniif nan sagada.
    Giiftii Keenya durboo Maariyaamiif nan sagada.Gooftaan keenya Iyyesuus Kiristooskan itti fannifame fannoosaaf nan sagada. Fannoonsaa humnaa keenya, humni keenya fannoosaatti kan nuu jabeessu fannoosaati, fannoonsaa furaa keenyadha. Fannoonsaa fayyisaa lubbuu keenyatti, warri Yihudotaa ni ganu nuti garuu itti amanna, warri itti amannus fannichaan ni faayyina, Fayyineerrasii!

  6. WAAQAYYOON KAN DEESSE KADHANNAA GIIFTII KEENYA MAARIYAAMII (Luq. 1:46-54)

    Maariyaamis jette, “lubbuun koo Waaqayyoon guddaa ni galateeffatti, afuurri koos Waaqayyo fayyisaa kootti ni gammaddi. Salphina garbittiisaa argeera’oo. Kunoo har’aa jalqabee dhaloonni hundumtuu na galateeffatu.

    Inni hojii guddaa naaf taasisera’oo. Maqaansaa qulqulluudha. Dhiifamnisaas warra isa sodaatanirratti dhalootaa hangaa dhalotaatti.Humnaas irreesaati murteesse. Akka yaada garaasaanitti warra oftuulan isaan bittinesse.
    Warra jabootas tessoosaaniirraa isaan salphise. Warra beela’anis arjummasaatiin qubse. Sooreesotas harka qulla hambise. Dhifamasaa akka yaadatuuf Israa’eel garbichasaa simate.
    Abbootti keenya Abrihaamiifi Sanyiisaaf akkuma dubbate.

    Hangaa bara-baraatti Ameen!
