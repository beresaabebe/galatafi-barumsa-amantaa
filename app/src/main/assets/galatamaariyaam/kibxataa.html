Kibxataa Kan Kadhatamu Gooftaa Keenya kan Deesse Galata Giiftii Keenya
Durboo MaariyaamKutaa Lammaffaa

  1. Goonfoo of-jajummaa Keenyaa, Jalqabni fayyina keenya, hundeen qulqullina keenyaa, nuu fayyisuuf nama kan ta’e kan ta’e jecha Waaqayyoo kan nuu deesse Maariyaamiin argame. Ergaa nama ta’een boodas Waaqa dhugaadha, Kanaafis durbummaadhaan deesseen. Dandeettiin dahumsashee dinqii ta’e, kan hin qoratamne dubbatamuuf kan hin danda’amnedha.
    Yaa qulqulleettii nuuf kadhadhu.

  2. Fedhii isaatiin, fedhii Abbaasaatiin, fedhii Afuura Qulqulluutiin dhufee nu fayyise. Durbummaadhaan cimtee kan jiraatte yaa Maariyaam , ulfinniifi galanni durbummaa kee olaanaadha. Waaqayyo waan si wajjin ta’eef ulfina aegatteetta. Samiitii hanga lafaa geesse Ergamoonnii Waaqayyoo yommuu irra yaabbataniifi bu’an riqaan Yaa’iqoob arge sidha.
    Yaa qulqulleettii nuuf kadhadhu.

  3. Biqiltuu Ibiddaan osoo hin gubatiin Museen arge sidha. Kunis, dhufee garaa kee keessa kan bule Ilma Waaqayyoodha. Ibiddi Waaqummaasaa foon kee hin gubne.
    Yaa qulqulleettii nuuf kadhadhu.

  4. Lafti qonnaa sanyiin itti hin facaane sidha; firiin jireenyaa si keessaa ba’e. Saanduqni yooseef oolchateefi Warqee (Inquu) ulfaataa kan keessatti argate sidha, Kunis Qorichaa keenya Iyyesuus Kiristoosidha, addunyaa kana keessattis Isa deesse.
    Yaa qulqulleettii nuuf kadhadhu.

  5. Gammachuu Ergamootaa kan ta’e Gooftaa kan deesse gammadi, Oduu haaraa raajotaa yaa qulqulleettii gammadi, Waaqayyo waan si wajjin jiruuf ulfina argatteeta’oo, gammadi Addunyaafi namoota hundaaf gammachuu kan ta’e jecha Ergamichaa fudhatteetta’oo gammadi, Addunyaa hundumaa kan uume Gooftaa kan deesse gammadi
    Yaa qulqulleettii nuuf kadhadhu.

  6. Dhuguma haadha Waaqayyoo waan jedhamteef gammadi, yaa fayyistuu Hewaan gammadi. Uumamtootasaa hundumaa kan jiraachisu harma kee waan isa hoosisteef gammadi. Haadha dhugoomtota hundumaa yaa qulqulleetti gammadi. Akka nuu kadhattuuf ija keenya gara keetti ol-kaasna.
    Yaa qulqulleettii nuuf kadhadhu.

  7. yaa Durboo, yaa qulqulleettii kan Gooftaa deesse nu fayyisuuf iccitiin dinqii siin kan raawwatame Mootii nuu deessetta, U=uumamtoota bifa adda addaatiin kan uume, waa’ee Waaqummaasaa dubbachuun nuuf hin danda’amuyyi haa callisnu.
    Yaa qulqulleettii nuuf kadhadhu.

  8. Gara tulluu siinaatti kan ba’e abboommii (seera) Museef kan kenne, fiixe tullicha hurii, aara, dukkanaafi qilleensaan kan uffise; warra sodaatanii dhabatanis sagalee faaggaatiin kan dheekkame qaama jecha Abbaati.
    Yaa qulqulleettii nuuf kadhadhu.

  9. Yaa tulluu gad-of qabinsaan dubbatte, kunis gara keetti kan bu’edha. Nama kan jaallate inni jijjiirama malee sirraa nama ta’e. Foon akkuma keenya dubbatuun dhuguma nama ta’e. Waaqayyo Afuura Qulqulluudhaan garaa kee keessa bulee dhuguman nama ta’e. Addaamiin fayyisuuf cubbuusaas dhisuf, samiirra isa jiraachisuuf baay’ina dhiifamaafi arjummasaatiin iddoosaa duraatti isa deebisuuf dhalate.

    Yaa qulqulleettii nuuf kadhadhu.

  10. Olaantummaa dubroo Maariyaam dubbachuuf namaaf hin danda’amu. Gooftaan ishee filateera’oo, Ifa itti dhiyaachuu hin danda’amne keessa kan jiraatu, inni dhufee ishee keessa bule; ji’a sagal garaashee keessa bule. Kan hin mul’anneefi hin qoratamne durbummaadhaan isa deesse.
    Yaa qulqulleettii nuuf kadhadhu.

  11. Raajichi Daani’eel kan arge, dhagichi tulluu dheeraarraa harka namaa malee ofiisaatiin kan konkolaate; jecha isa Abbaa irratti ba’eedha; dhufees sanyii dhiiraa malee durboo irraa dhalatee nu fayyise.
    Yaa qulqulleettii nuuf kadhadhu.

  12. Akkuma fixa(qariqara) qulqulluu taate, hundeen amantaasidha. Qajeeltuu kan taate amantaan abbootii keenyaa qulqullootaa sidha. Waaqayyoon kan deessefi durbummaadhaan kan jiraattu yaa qulqulleetti jecha Abbaa nuu deesse. Iyyesuus Kiristoos nu fayyisuuf dhufe.
    Yaa qulqulleettii nuuf kadhadhu.

  13. Yaa ulfaattu Gooftaa kan deesse sidha. Jecha hin mul’anne kan baatte haati ifichaa sidha. Erga isa deessee boodas durbummaadhaan jiraatteetta’oo. Galata guddaadhaan si galateffatu.
    Yaa qulqulleettii nuuf kadhadhu.

  14. Jecha Abbaaf haadha kan taate yaa qulqulleetti durboo Maariyaam, dubbii waa’ee keef dubbatamu dubbachuu kan danda’u arraba kamiidha? Teessoo Mootii Kiribeel baatan taate. Yaa ulfaattuu si galateeffanna, yaa haadha Gooftaa keenyaa Iyyesuus Kiristoos gugee gaarii maqaa kee dhalootaa dhaloootatti ni waamna.
    Yaa qulqulleettii nuuf kadhadhu.

  15. Yaa Maariyaam haadhaafi tajaajiltuu Waaqayyoo kan taate gammadi. Isa ati haammatte Ergamoonni ni galateeffatu. Kirubeelis sodaadhaan isaaf sagadu, Suraafeelis utuu walirraa hin kutin kaattuu saanii diriirsanii “Mootiin galateeffatamu isa kana baayina dhifamasaatiin cubbuu addunyaa balleessuuf kan dhufe isa kanadha” jedhu.
    Yaa qulqulleettii nuuf kadhadhu.
