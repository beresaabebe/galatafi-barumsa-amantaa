Roobii Kan Kadhatamu Gooftaa Kan Deesse Galata Giiftii Keenya Durboo Maariyaam
Kutaa Sadaffaa

  1. Lafa lammaffaa kan taate yaa samii; Ergamoonni Samii “kan eebbifamte” siin jedhu; Dubroon Maariiyaam balbala baha biiftuutii. Misirroon ishee qulqulluu kan tahe mana cidhaa qulqulleettiidha. Abbaan samiirra ilaalee kan akkake hin argannee, kanaafis Ilmasaa tokkiicha garakeetti ergee sirraa nama ta’e.
    Yaa quulqulleettii nuuf kaadhadhu.

  2. Waaqa kan deesse yaa Giiftii keenyaa dhaloonni hundumtuu siin galateeffatuu. Yaa magaalaa Waaqayyoo raajonni dubbii dinqii tahee kan siif dubbatan, mana bultii Qulqullootaa taateettaa .Mootonni lafaa hundumtuu ifa keetiin deemuu. Uummataaafii Raajonni isaanii ifa keetiin deemuu.Yaa Maariyaam dhalloonni hundumtuu si galateeffatuu.i Isa sirraa dhalateefis ni sagaduu, ni galateeffatuu.
    Yaa quulqulleettii nuuf kaadhadhu.

  3. Biishaan bokkaa kan sirraa argame duumessi dhugaa sidhaa. Abbaan mallattoo Ilmasaa si taasise; Afuurri Qulqulluun sirraa bulee, humni Waaqayyoos si golgee, Yaa Maariyaam bara baraan kan jiraattuu jecha Abbaa kan ta’ee Ilma nuuf deesse, dhufees cubbuurraa nufayyise .Gammachuudhaan missiraachoo kan himtee Yaa Gabri’eel Ergamaa, ulfinni siif kenname guddaadha, gara keenyatti kan dhufee Dhaloota Gooftaa nutti himte; Dubroo Maariyaamiiniis “kennaan kan guutamte Waaqayyoo si wajjiin waan jiruuf gaammadii” jettee missiraachoo itti himte.
    Yaa Qulqulleettii nuuf kaadhadhu.

  4. Ayyaanaan (Kennaan) guutamte, Afuurri Qulqulluun sirra bulee; humni Waaqayyoos si golgee, Yaa Maariyaam dhugumatti Qulqullicha deesse,addunyaa hundumaa kan fayyisuu,dhufee nu fayyisee.
    Yaa Qulqulleettii nuuf kadhadhu.

  5. Arraabni keenya hojii Dubroo ni galateeffataa. Gooftaafi Fayisaan keenya Iyyesuus Kiristoos ,magaalaa Daawwitiitti isheerraa waan dhalateef, Yaa namootaa kottaa Waaqa kan deesse Maariyaamiin har’a haa galateeffannu. Haadhaafi Dubroo lamaanuu taateetti hoo. Mudaa kan hin qabneefi jechi Abbaa dhufee sirraa nama kan ta’e yaa qulqulleettii Dubroo gaammadii, qaaniin kan sitti hin jirree, Qulqulluufi xuuraa’inni kan sirra hin jirree, yaa mudaayii qulqulluu gammadii. Nama isa duuraa Addaamiif jecha, Addaam lammaffaa kan ta’e Kiristoosiif mana bultoosaa kan taate, Yaa dubbattuu gannataa gaammadii. Abbaasaa wajjin kan jiraatu, tokkicha isaa kan baaatte gaammadi. Inni dhufee sirraa nama kan ta’e, yaa qulqulleettii mana cidhaa gaammadi. Ibiddii Waaqummaa kan si hin gubne yaa lalistuu Phaaxoos gammadi. Waaqayyoo Kiruubel irra taa’u, Fooniin kan baatte, tajaajiltuufi haadha, Dubroofi Samii gammadi. Kanaafuu Ergamoota Qulqullaa’oo kan ta’an wajjin gammachuu guuddaatiin gammannee haa galateeffannuu. Samiirrattiis Waaqayyoof Galannii haa ta’u, lafarrattis araarri haa ta’u haa jennu. ulfinaafi galata Gooftummaasaa kan qabu inni waan si jaallateefi.
    Yaa Qulqulleettii nuuf kaadhadhuu.

  6. Ulfina Qulqullootaarra ulfina Maariyaamiitu caala. Jecha Abbaa simachuuf qophooftee waan argamteef , Ergamoonni kan sodaatan tattaafattoonni isaa Samiirratti kan galateeffatan Dubroon Maariiyaam garaasheetti baattee. Isheen Kiruubeliin ni caalti, Suraafeeliinis ni caalti, qaama sadan keessaa isa tokkoof mana bultii waan taateef. Biyyi Raajotaa Iyyaruusaaleem ishee kana. Qulqulloota hundumaaf mana bultii gammachuu isaaniitii. Uummattaa gaaddidduu dukkanaafii du’aa jala jiraataniif ifni guddaan ni ifeef. Qulqullootarra bulee kan jiraatuu Waaqayyo nu fayyisuuf Dubroo adda taaterraa nama ta’e. Kottaa dinqii(raajii ) kana ilaalaa. Iccitii nuuf mul’ateefis Galata galchaa; Nama ta’uu kan hin mallee nama ta’eeraa; Jechis Foon uffateera. Foon durdurummaa hin qabne kan duriifi dursa ta’e; Waaqummaan barri hin lakka’amneef baratu lakka’ameef. Kan hin beekamne ifa ba’ee.Dhugumatti ilmi Waaqayyoo nama ta’e. Duraan kan ture har’as kan jiru, yoomiyyuu kan jiratu Iyyesuus Kiristoos akka isaaf sagannuufi galateeffannuuf inni Waaqa tokkodha.
    Yaa qulqulleettii nuuf kadhadhu.

  7. Raajichi Hizqi’eel waa’ee ishee akkana jedhe raggaasise, “Balbala Dinqii kan ta’e, cufa jabaadhaan kan cufame gara bahaan nan arge jedhe, Gooftaa jabaa malee garasheetti galee kan bahe hin jiru.” Yaa qulqulleettii nuuf kadhadhu.

  8. Karris balballis fayyisaa keenya kan deesse Dubroodha. Ergaa is deessen boodas akkuma duraanii durbummaadhaan waan jiraatteef. Harka diina keenya isa dhiifama hin qabne jalaa dhufee kan nu baase Gooftaa Keenya kan nuuf deesse yaa Durboo firiin garaa keetii kan eebbifamedha. Atis kan eebbifamteedha; Waaqa dhugaa ulfaataa kan ta’e biratti ulfina waan argatteef, biyya lafaa kanarra kan jiraatan hundumaa caalaa ulfinniifi galanni siif ni ta’a. Jechi Abbaa dhufee sirraa nama ta’e. Nama wajjiin ni deddeebi’e. Innis Gooftaa dhiifama namaaf taasisuuf nama kan jaalalatu waan ta’eef dhufaatii addaa ta’een dhufee nu fayyise. Yaa Qulqulleettii nuuf kadhadhu.
