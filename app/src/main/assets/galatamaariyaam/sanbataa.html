Giiftii Keenyaa Durboo Maariyaam kan Sanbata Duraa Kadhatamu

Kutaa Jahaffaa

  1. Yaa Maariyaam ati Qulqulleettiidh, eebbifamtuudhasGooftaa harka keen kan qabdeen yaa Dubroo ati waan hundumaan kan eebbifamtedha.
    Uumamtoonni hundumtuu, “Yaa Qulqulleetti nuuf kadhadhu.” jedhanii sagalee guddaadhaan si wajjin gammadu. Kennaan kan guutamte yaa Durboo gammadi,
    Waaqa biratti ulfina waan argattef gammadi, Waaqayyo si wajjin waan jiruuf gammadi.
    Simboo kan qabdu yaa Durboo olaantummaa kee ni galateeffana. Fayyinni keenya firii garaa keetiin waan argameef yaa Maariyaam nuti akkuma ergamaa Gabri’eelii,
    galata siif dhiyeessina. Gara Abbaasaa gara Waaqayyootti nu dhiyeesse.
    Yaa Qulqulleetti nuuf kadhadhu.
  2. Afuurri Qulqulluu waan sirra buleef, humni Waaqaas waan si golgeef ati akkuma mana cidhaa qulqulluudha. Yaa Maariyaam jechaa Abbaa
    bara baraaf jiraatu dhufee cubbuurraa kan nuu fayyise dhugumatti nuuf deesse.Yaa Qulqulleetti nuuf kadhadhu
  3. Sanyii Daawit keessaa kan argamte si’idha. Fayyisaa keenya Iyyesuus Kiristoosiin fooniin waan deessef addunyaa kana dursee kan ture Abbaarraa kan dhalate jechi foon nama uffatee WaaqummaasaaYaa qulqulleetti nuuf kadhadhu.
  4. Yaa Maariyaam mudaa osoo hin qabaatin Gooftaa kan deesse, aduun dhugaas si keessaa waan baheef ati lafa lammaffaa Sami taate. Akkuma raajonni raajii dubbatanitti sanyii dhiiraa malee otoo hin jijjiiramiin Gooftaa deesse.Yaa Qulqulleetti nuuf kadhadhu.
  5. Kan filataman keessaa adda kan taate, Tsillaatiin kakuu kan si keessa jiru, leemmanni Warqeefi nyaata dhokate kan baatte kan si keessa jiru, godoo /dunkaanni/ dabtaraa sidha. Nyaata kan jedhame kunis garaa Dubroo Maariyaam
    keessa kan bule Ilmaa Waaqayyoodha. Jechi Abbaa ishee irraa dhalatee nama ta’e. Dhufee kan nuu fayyise Mootiin samiifi lafa addunyaa keessatti deesse.Haati Kiristoos hoolaa dubbattu gannataa kan taate ni gammadi.
    Bara baraaf kan jiraatu Ilmi Abbaa dhufee nu fayyise.Yaa Qulqulleetti nuuf kadhadhu.
  6. Haadha Mooticha Kiristoos jedhamte.
    Erga isa deeseen boodaas durbummaadhaan jiraatte.
    Iccitii dinqii ta’een Gooftaa deesse.
    Kanaafuu jijiirama malee si eege.Yaa Qulqulleettii nuuf kadhadhu.
  7. Riqaan Waaqayyo irra taa’ee yaa’iqoob arge sidha. Karuma hundumaan qoratamtee bira kan hin ga’amne garaa kee isa qulqulluu ta’e bira baatteen. Nu fayyisuf jedhee sirraa nama kan ta’e Gooftaa keenya Iyyesuus biraatti araarsituu nuuf taate.Yaa Qulqulleetti nuuf kadhadhu.
  8. Yaa galma eebbifamtuufi qulquluu kan taate, yaa Yaa Maariyaam kunoo Gooftaa baay’ina dhifamasaatiin addunyaa kan uume, hundumaa fayyisuuf si’i irraa dhalate. Inni arjaafi nama kan jaallatu waan ta’eef haa galateeffannun.Yaa Qulqulleetti nuuf kadhadhu.
  9. Cubbuu tokko malee Dubroo kan taate, kennaan kan guutamte, yaa Giiftii keenyaa gammadi. Ati ulfina addunyaa hundumaafi Xoofoo qulqullooftuudha. Yaa Maariyaam ati ibsa hin dhaamne, Mana qulqullummaa hin digamnedha. Hirkoo qulqullootaa, uleen amantaa hin jijjiiramne sidha. Cubbuu keenya akka nu dhiisuuf Ilma kee arjaafi fayyisaa ta’e nuuf kadhu.
    Yaa Qulqulleettii nuuf kadhadhu.
