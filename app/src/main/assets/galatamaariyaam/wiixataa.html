Efreem qulqulluun kan barreesse guyyaa Wiixataa kan kadhatamu
Gooftaa Keenya Kan Deesse Galata Dubroo Maariyaam
Kutaa isa jalqabaa

  1. Gooftaan, kan lapheensaa gadde Addaamiin bilisa baasuu
    fi iddoosaa duraatti deebisuuf jaallate.
    Yaa qulqulleetti nuuf kadhadhu.

  2. Dubroo Maariyaam irraa sanyii dhiiraa malee fooniin dhalatee nu fayyise. kan Seexanni dogogorse Hewaaniin “ciniinsuu kee nan sitti
    baay’isa”     jedhee Waaqayyo itti murteesse. Nama jaallachuusaatiin garuu bilisa ishee baase .
    yaa qulqulleettii nuuf kadhadhu.

  3. Nama kan ta’ee fi jechi nurra bule, Iyyasuus kiristoosi dha.
    Kabajasaa isa Abbaa isaaf tokko ta’e ni argine. Cubbuu keenya nuu dhisuuf jaallate.
    Yaa qulqulleettii nuuf kadhadhu.

  4. Raajichi Isaayyaas, Afuura qulqulluutiin iccitii Amaanu’eel arge. kanaafis, “Mucaa nuuf dhalate ilmis nuuf
    kenname” jedhee dhaggeesisee dubbate.
    Yaa qulqulleettii nuuf kadhadhu.

  5. Waaqayyo addunyaa kana waan jaallateef yaa namaa gammadi ;
    Ilmasaa tokkichatti kan amanan hundumtuu hanga
    bara baraatii akka fayyaniif isa dabarsee nuuf kenneera waaqayyos irreesa nuuf erge .
    yaa qulqulleettii nuuf kadhadhu.

  6. kan duraan jiru ture, ammas kan jiraatu, kan dhufe lammaffaas kan dhufu jechi,
    Iyyasuus Kiristoos, otoo hin jijjiiramin gonka nama ta’e. Ilmi tokkichi hojiisaa
    hundumaatiin addaan hin baane, Waaqummaan jecha Waaqayyoo tokkodha malese.
    Yaa qulqulleettii nuuf kadhadhu.

  7. Yaa Beetaliheem biyya Raajotaa gammadi , Addaam inni lammaffaan, Kiristoos
    , so keessatti dhalateera’woo;
    Nama is a duraa Adaamiin si’oolii gara gannataattii deebisuuf, “yaa Addaam lafa turte gara lafaattis ni deebita”
    jedhee murtii du’aa itti murteesse irraa kaasuuf dhalate.
    Iddoo cubbuun baay’ee jirtutti kennaan Waaqayyoo in baay’atti.
    Yaa qulqulleettii nuuf kadhadhu.

  8. Dhaqni namoota hundumtuu ni gamaddi “samiirratti Waaqayyoof galanni haa ta’u,
    lafarratiis nagaan dhala namaafis haa ta’u hangs inni eeyyameetti” jedhanii Ergamoota samii
    wajjin ni galateeffatuun. Abaarsa duraan ture balleesseera, gorsa diinaas jalaa diige, Addaamiifi Hewaaniif xalayaa du’aa tarsaaseef.
    Biyya Daawititti kan dhalate fayyisaan keenya Iyyesuus Kiristoos addaamiifii hewaniin bilisa base.
    Yaa qulqulleettii nuuf kadhadhu.

  9. Namoota addunyaa kanarra jiraatan hundumaaf kan ibsitu ibsaan dhugaa , jaalala namaaf jettee gara addunyaa kanaatti kan dhufte.
    Uumamni hundumtuu dhufaatii keeti ni gammadu. Addaamiin dogogorarraa deebisteenii, Hewaaniinis
    hidhaa du’aa jalaa baasteenii
     , Afuura ittiin dhalannu nuuf kennite, Ergamoota wajjin si galateeffanna.
    Yaa qulqulleetti nuuf kadhadhu.
