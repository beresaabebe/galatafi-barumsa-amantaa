Kutaa 2ffaa
  1. Inni Iyyesuus Kiristoos warra du’an irraa adda ba’ee ka’uudhaan, gara si’ool bu’ee cufaa sibiilaa si’oolii cabsee du’u isaan du’a ni balleesse, ni barbadeesse, gaafa guyyaa sadaffaas, du’oota irraa adda ba’ee erga ka’ee booda, yaa abbaa, jechi Waaqayyoo akkana kan raawateefi kan hariifatedha. Sin galateeffadha jedhee, jechi foonii Abbaasaa galateeffate.
  2. Yaa Abbaa, kanan si galateeffadhus, afaan fooniin miti, sobaafi dhugaan kan ittiin dubbatamu afaan kanaanis miti. akkeenyaa fooniirraas kan argamu, jevha kanaanis miti. Mootiin sirraa argame hundumaa biratti kan ittin beekamu jecha kanaan malee.
  3. Kanan waan ta’eefis garagurra foonii kan hin seennedha, addunyaa kanarra kan jiru jecha dubbatamuun mitihoo.
  4. Sirraa argamtee, jecha addunyaan hundi ittin umamte, jiraachumaa keefi jecha jiraachumaa koon kan jiru, afuurri qulqulluun jiraataa ta’ee kan dubbatu, jecha kanaanidha.
  5. Yaa Abbaa, Atumti kan jaallatamtu, atumti kan galateeffamtu. ati abbaadhahoo, jaboo kabajamtoonni kan ta’an amantoonni lammiiwwan kee hundii gara keetti gatii (horii) ittiin kadhatanidha.
  6. Yaa Abbaa, jaallewwanni siin ni waamu, gargaaraa siin ni waammatu.
  7. Yaa uumaa hundaa, hundi jecha keen uumameerahoo, hundi siif ni bitama. Addunyaa dura kan turte, si qopharraan kan hafe Waaqayyo hin jiruhoo dhuguma bara baraan kan jiraattu si qofaadha.
  8. Egaa tikseen isaa yookiin barsiisaan isaa, dubbii iccitichaa akka ibsuuf jecha, gara abbaatti otoo hin kadhatiniif hin ba’in dura akka beekitaniif akka hubataniif, tiksichi karra mana mootummaa samii ittiin seenan akka beekuuf ni barbaachisa jedhee Iyyesuus ni dubbate.
  9. Qulqulluu foonsaafi kabajamaa dhiiga isaan kan itti firooman, yeroo kadannaa qiddaasee (waaqeffannaa) maqaasaa kan waaman eenyuu akka ta’e laphee isaaniin yoo beekan, yoo hubatanis ni gargaaramuhoo.
  10. Dabalees amantoota haaraan, isin yaa obbolota koo gara Waaqayyootti dhiyaattaniittu, isa qofaa amantanii jabatanii akka jirtan beektaniittuhoo ittiin haa jedhan.
  11. Ishee akka isinii kennuufiif, amantaa isa inni dubbates gatii akka taasifattaniifis jecha. Kunis iji kan hin argine gurris kan hin dhageenye laphee namootaan kan hin yaadamne, Waaqayyo warra jaallatu qofaaf kan qopheessedha.
  12. Museefi raajonni qulqulloonni kan biroon, Amantaan gaarii (horii) Waaqayyoo ta’uusaa nutti himaniiruhoo, Nu barsiisaniiruhoo, nutis kabajinni galanni jabinni bara baraan isaaf kan ta’u ta’uusaa dhuguma ni amanna bara baraan ameen.
