Kutaa 1ffaa
  1. Maqaa Abbaa, Ilma, kan Afuura Qulqulluu Waaqayyoo tokkoon ameen.
  2. Amantootaaf dhokisaa kan ture, mul’ataa Kiristoos booda garuu, sera Qurbaana Qulqulluu dura kan dubbatamu, kan dubbifamu, baru misi amantaa
    kadhannaan galata Waaqummaa barumsa Hibu’aatiin Kanadha. Kunis kitaaba waadaa irraa kan argamu yemmuu ta’u, waadaa torbanuu gidduutii isa tokkodha.
  3. Waa’ee barumsa Hibu’aatii garuu akkanatti dubbanna, akkanatti barsiisna, akkanas jechaa ni kadhanna.
  4. Addunyaa dursee kan ture, har’as kan jiru, Addunyaas dabarsee bara baraan kan jiraatu, nama ta’uuf kan dhufe. Dabalees addunyaa dabarsuuf kan dhufu.
    Foon uffateen dhala namootaaf jedhee kan dhukkibsate, du’a isaatiin dhibee du’aa kan balleesse, kan awwalamee, gaafa guyyaa sadaffaas warra du’ani irraa adda ba’ee kan ka’ee inni Iyyesuus Kiristoosiidha.
  5. Jireenya Afuura Qulqulluutiin, akkeenya Addaamii, akkeenyaa isaa taasifatee nama yoo ta’e iyyuu, otoo hin jijjiiramin fakkenyaa Waaqummaa isaatiin jabaatee bara baraan kan jiraatu inni Iyyesuus Kiristoosiidha.
  6. Foon du’u Waaqummaa isaa wajjin tokko taasifate, akkeenyaa waaqummaa isaatii kan taasifate, kan kabaje, du’as fooniin injifatee gara samiitti kan ol ba’e, bara kuma shaniif dhibba shan nu namoota irratti leenjifamee kan ture,
    kiyyoo taayitaa diyaabiloosii kan kukkuteef humna isaa kan inni ittin mul’isu dhiphina bultee isaatii kan balleesse inni Iyyesuus Kiristoosidha
  7. Taayitaa dhalacuu isaatiin dandeetti kan qabu Ilmi Waaqayyoo, durboo Maariyaam irraa foon fudhatee lammii isaa baasuuf gara si’oolii bu’ee yemmuu argu, duukana kan uffate diyaabiloosii ni sodate ni na’ees.
  8. Gorsa adda hin baaneef gargar hin baane, fedha tokkittin Abbaa isaa wajjiin samii kan uume, inni abbaa isaa wajjin Gonfoo ergamootaati. Hogganoota Ergamootaatiifis humna isaanii kan ta’e, fedhii abbaa angootaa, bittotaafis gadi fageenyiisaanii bara baraan jabaatte, kan jiraattu, mootummaa kan qabidduuf bittoota dhugomotaa inni Iyyesuus Kiristoosidha.
  9. Kan hin qoratamne sammuun Abbaasaa isadha. Ogummaan abbaa isadha. Humni abbaa mirgi harka isaa jabaan harkisaa inni Iyyesuus Kiristoosidha.
  10. Dursee kan ture ta’uu isaa amannee, fooniinis mul’atte argama fayyinna keenyaa ta’uusaaf isaaf ni bitamna. Waan badiirraa kan nu dhoksu, yeroo rakkinaa kan nu gargaaru, yeroo hubamnu kan nuu ga’u Gooftaan, gatii guutuu kan kennu, da’oon keenya, tolummaan kan nuu tiksu, gargaarsasaan kan nurraa adda hin baane, amantaan kan nu jabeessu, diinni kan ija itti hin jabaanne, daangaan jabaataan keenyaa argama gammachuu, balballi bahaafi lixa keenyaa, jireenya kan hiru hirbaanni keenyaa qabana afuuraa kan nuuf kennu dhugaatiin keenya inni Iyyesuus Kiristoosidha.
  11. Kan dhalate malee kan hin uumamne, kan murteessu kan hin jirre, inni garaa dubrootti kan murtaa’e, cubbuun otoo irra hin jiraatin, cubbamaa jedhamee cubbuu keenya baacu n fannoorra kan ba’e, kan du’ee inni jiraataa akka ta’e kan ittiin amannu, abbaasaa biraa adda kan hin baane Ilmi abbaa inni Iyyesuus Kiristoosidha.
  12. Oli olaantummaasaan kan jiru tokkichummaa abbaasaarraa otoo adda hin ba’in, samiiraa gadi bu’ee, foonsaatiin gidiraa fudhateen kan nu fayyise, badhaadhina qulqullinaa, badhaadhina jireenyaa kan ta’u foonsaafi Dhiigasaa isa fayyisu kan nuu hire, bishaan cuuphaatiin cubbuurraa bilisa kan nuu taasise inni Iyyesuus Kiristoosidha.
  13. Inni bara baraan irra bulee kan jiraatu yemmuu ta’u, laphee namoota isa sodaataniirra bulee ifa beekumsaa kan ibsu, daandii ammeenyaa seexanni irra deemu hundaarraa akka fayyinuuf ogeesota kan nu taaisu, lubbuu keenyas galaana si’oolirra kan ceesisu, bara hundaa dura kan ture, hundi keenya Gooftaa nutti itti amannu inni Iyyesuus Kiristoosidha.
  14. Addunyaa dura kan ture, Waaqayyo abbaa wajjiin bara baraan jabaatee kan jiraatu inni sammuu aarturraan (dallanturraan) kan ka’e dogogorri kan fide qormaataafi sammuu dhabuudhaan addunyaan badiidhan qabamee, yemmuu dhiphatu gaafa argu, akkeenyaa namaa fayyisuuf jaallatee, Ergamoota samiirraa dhokatee gara dubrootti murtaa’uun gara isaatti nu dhiyeesse, yaada badii hundaa nurraa fageesse, seexannis akka hin beekne taasise.
  15. Jecha hin qoratamneefi hin jijjiiramne, foon jijjiiramu uffate, jijjiiramuun kan ta’uufiif foon kan hin jijjiiramne taasise. Kanaafis kan waaqummaasaa wajjin tokko taasifte, foon Addaam isa du’uunis ni mul’atte. Akkeenyaa hin jijjiiramne kan qabu jechi, akkeenyaa jijjiiramu kan qabu fooniin du’ee badiin kan qabaman fayyise.
  16. Abboomiisaa isa dhugaa kenneef, kunis dubbii Gooftummaasaatii kan dubbatu jecha wangeelaati. Namootarraas irra bulummaa seexanaa ni balleesse. Egaa du’a Gooftaa keenyaan jireenya akka argannuuf beekumsa dhabiisarraa gara beekumsaatti ceenee, mootummaa samii ishee hin darbinerra akka jiraannuuf dadhabbirraa haa damaqinnu.
  17. Nama yoo ta’e iyyuu inni Kiristoos ilma Waaqayyootti, dhaloota foonii dhalachuuf kan dhufe, kan jalqabaa inni Addaamirraa kan argame foonsaa du’u gatii taasifate.
  18. Dursee Raajotaan kan beekame, labsii duuka bu’otaanis kan mul’ate uumamni hundinu kan itti amanan. Ragaa Waaqayyo Abbaan kan mul’ate, jecha Ergamootaanis kan galatoome nama kan ta’e inni Waaqayyo dhugaadha.
  19. Nuun fayyisuuf fannnorratti fannifame, fannoosaanis jireenya arganne. Qoodi fayyisaan keenyaaf jabeessan keenya isadha. Kunis Waaqayyoorraa kan hin fagaanne, yoomi yoomiyyu kan adda hin baane, yeroo hundumaa kan hin madaalamne baayina tolummaasaatiin dubbachuun kan hirrawwatamne, gammachuun kan irra jiru iccitii gadi fagoodha.
  20. Kanaafi arraba gararraafi gajallaa kanaan iccitii kana akeekkaffatanii dubbachuun hin dandaa’amu. Amantoonni qoratanii baru kan hin dandeenye iccitii gadi fagoodha. Kan hin qoratamne yoo ta’eyyuu garuu isaaniif kan mullate ta’e. Mul’achuunsaas ijaan mul’achuuf miti, lapheen yaadamuuf malee. Kanaaf akka galateeffannuuf jecha nuti itti ofi jajnu kanadha.
  21. Isa amanuudhaan amantoonni cuubaman ga’oodha (jaboodha) kan mul’atu irra kan hin taane kan dubbatamuu hundaarra qaama isanii dhuguma ni fayyisan. Sammuu gaarii kanaan jaboo kan taatan, ofii keessan qoradha, afaan gurra keessaniis badii dhaga’uurraa afaan ija keessaniis badii ilaaluurraa ittisaa.
  22. Iyyesuus Kiristoos kan jaallatu akka beektaniif jecha, iccitii fayyina keessaniis akka hubattaniif, isin Kiristoositti kan ofi jajjan (amanamtan), kabajamoon dhiiraafi dubroonni, seera isaa isiniif kan jabeesse afuura qulqulluus isinirra kan bulche isinidhahoo, hojii lubbuu hojjadhaa, keessumatti, keessi qaama hundaa kana haa hubatu. Kiristoos tokko taasisee gara abaasaatiitti nu dhiyeessuu waan fedheef booda gara si’ooliitti bu’ee, lubbuu du’ootaa jireenyaaf kan booji’e inni Iyyesuus Kiristoosidha.
  23. Lubboota si’ool jiran baasuuf, Kiristoos fooniin gara si’oolii yemmuu bu’u gaafa argu, duutni sagalee guddaan akkana jedhee iyye. Foon nyaatamuusaarraa haaromsee, jijjiiramuusaarraas fayyisee kan tokko taasifate (irra bule) kun eenyuudha? Foon taayitaa koo jala jiru uffatee, balaqeettii ifa akkeenyaa isaarra jiruun kan na injifate kun eenyuudha?
  24. Gara badiitti buutee na fudhachuuf, taayitaa koo jala otoo jiruu, taayitaan kan kennameefiif kun eenyuudha? Inni garuu kan samiiti, akkeenyaa jijjiirama kanaan kan dhalate kun eenyuudha? Inni garuu kan hin jijjiiramnedha.
  25. Seera koo irraa adda kan ta’e kun eenyuudha? Ana bira kan turan ormoota lammii koorraa adda baasee, kan boji’uu kun eenyuudha? Humnaa ibidaan du’aa wajjin kan hiriiru kun eenyuudha? Duukkana kan injifate kun eenyuudha? Duukanaan kan guutaman awwalichiifi si’ooliin qabanii hambisuu kan hin dandeenye kun eenyuudha?
  26. Bifa keessumaa kanaan, kabaja adda ta’ee kanaan kan jiru. Isarratti raawachuuf kanan barbaadu hundumaa taayitaan isaa kan na ittisuu kun eenyuudha?
  27. Cubbuu malee kan du’e kun eenyuudha? Balaqeetti ifa humna isaan dura ta’aa duukana jaamaa kan taasise kun eenyuudha? Lammii koo akka ani hin binneef kan nan dhiisne kun eenyuudha? Haa ta’u malee lubbuu naaf kennamanii turan hunda gara samiitti ni baasahoo.
  28. Dhibeedhaafi du’aan akkenyaa jijjiiramuu wajjin tokko ta’ee akka inni irra hin bulleef kan na ittisu taayitaan kun maalidha? Itti bubu’un kan naaf hin dandaa’amne kun eenyuudha? Ifi kan marsu, hirrinnis kan irra hin jirre kun eenyuudha? Lammii isaa akkan hin balleesineef taayitaa koo irraa kan na buusu, isa irrattis leenjifamee taayitaa ittiin miidhu kanan hin qabne kun eenyuudha?
  29. Kunis Iyyesuus Kiristoos isa fannifameedha, gara harka bitaa kan jiru gara harka mirgaatti kan ittin ca’e, gadi kan jiru kan ittiin ol ba’e boodasaa kan jiru gara fuula duraatti kan ittin durse. Lubbuuwwata si’oolii jiran olantummaan gara ganataatti kan ol baaste, taayitaa isaanirraa bu’anii kan turan gara kabajaa isaanii isa duraatti kan deebise inni Iyyesuus Kiristoosidha.
