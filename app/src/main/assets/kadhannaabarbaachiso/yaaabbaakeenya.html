• Yaa Abbaa keenya Samiirra kan jiraattu maqaan kee haa qulqullaa'u. Mootummaan kee haa dhufu jaalalli kee samiirratti akkuma ta'e akkasumas lafarratti haa ta'u. Kan nu ga'u soora keenya ara'aaf ga'u nuuf kenni, yakka keenyas nuuf dhiisi, nutis kan nu yakkaniif akkuma dhisnuuf, qorumsatti nu hin galchiin; hamaarraa nu oolchi malee, Mootummaan kan keetti' oo humnis galannis bara-baraan Ameen!

  • Nagaa Ergamaa Gabri'eel qulqulluutiin Yaa Giiftii keenya Maariyaam nagaa siin jenna. Yaada keetiin durboodha, foon keetiinis durboodha. Yaa hadha Waaqayyoo isa mo'aa siif galanni haa ta'u. Ati dubartoota keessaa kan eebbifamteedha firiin gadameessa keetis kan eebbifameedha. Gammadi yaa ayaana qabeetti Waaqayyoo si wajjin jira'oo Gooftaa keenya Iyyesuus Kiristoos irraa dhiifama nuuf kadhadhu, cubbuu keenya akka nuuf dhiisuuf. Hangaa bara-baraatti Ameen!
