Galata (Lixoonii) Guyyaa Gaafa Dilbataa (Sanbataa Kiristiyaanaa)

Uumaa Gooftaa abbaa dhiifama keenya daballee ni kadhanna. Abrihaamiin kan fayyiste sidha, mootota yeroo waraane Yisihaaqiin qalamuu itti dhufe irraa kan fayyise, Yaa’iqobiiniis karaatti kan geessite dhagaa bakka dhaabbatetti halkan mul’ataan kan itti mul’attee, Yoseefin biyya ormaatti kan ol ol qabde. Yaa Waaqayyo uuma.

Yaa Abbaa kenyaa jedhaa kadhadhaa.

Dhugumadha yaa uuma. Nohiin bidiruudhaan kan ceesiste sidha. Daani’eeliin afaan leencaarraa kan fayyiste, Soosinnaas harka sobdootaarra kan fayyiste. Musee ulfina guddaa wajjin gara gibtsitti kan ergite, lammii kee israa’eelota guyyaa uriidhaan halkan guutuu ifa ibidaan kan gaggeesite, raajota irra bulee raajii kan dubbate duuka buutota kee irratti surraa kee kan ibsite sidha. Mana tokko keessatti walitti qabamanii bakka jiranitti afuura qulqulluun akka ibiddaatti isaanirra bu’e, qooqa biyya adda addaattiinis ni dubbisaan. Samiif lafa kan uumte ati afuura kee nuuf ergi. Sa’a hundaaf yeroo hundaaf tokkichummaa keenya nuuf eebbisi nu kabaji kan dursuuf kan itti aanu kan hin qabne tolummaa Ilma keetiin isaaf afuura qulqulluu wajjiin ulfiniifi galanni haa ta’uuf. Har’aas yeroo hundumaa hanga bara baraatti ameen!
