Galata (Lixoonii) Guyyaa Gaafa Jimaataa

Yaa Gooftaa, kadhannaa tokkoon tokkoo nama yaadadhu, yaa Gooftaa tokkoon tokko namootaa gadameessa haadha isaaniirraa kaasii yaadi namoota amantii dhugaadhaan jiraatan hundaa yaadadhu. ubroota isaanii eegi, dargaggoota isaanis qajelchi maanguddoota isaanis jabeessi. Dhiphinaan kan rukutaman (reebaman) fayyisi. Ari’atamuudhaan kan bittinfaman walitti qabi. Karaa warra deemanii wajjin eebba keetiin deemi. Dhiphinaan kan haccuucaman fayyisi. Lolaaf wal morkiidhaan, bita jabaa jalas kan jiran tolummaa keen yaadadhu. Addunyaa kana qabdee kan jirtu yaa Waaqayyoo uuma tolummaa keen nuu yaadadhu jedhaa kadhadha.

Yaa Abbaa keenyaa jedhaa kadhadha.

Dhuguma yaa uuma, maanguddota kan gargaartu, abbaaf haadha kan hin qabnee ijoollee iyyeesotaa dhiphina irraa kan fayyistu, kan dhukkubsataniif qoricha, karaa kan deemaaniif karaa gaariin sidha. Yaa Waaqayyo ati kadhannaa isaaf jireenya isaa, yaada mana isaatiis qoratte ni beekta. Yaa Gooftaa biyyattii kana, biyyoota adda biraas amantii keettis jabaatanii kan jiraatan fayyisi. Dhugoomtota qulqulloonni socha’uurraaf beelarraas akkasumas lammii biraarraa weraara isaanitti dhufu irraas fayyisi. Ifaa jaallatamuuf bareedu, beekumsa kee nuu ibsi. Shakkii mana qulqullummaa keerraa fageessi. Ijoollee galata ifaa kan ta’an Ergamootaa wajjin tokko nu taanee Abbaa Ilmaafi Afuuraa qulqulluu akka galateefannuuf, galata nuu laadhu, har’aas yeroo hanga bara baraatti ameen.
