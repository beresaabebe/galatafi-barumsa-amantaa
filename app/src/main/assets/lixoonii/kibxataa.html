Galata (Lixonii) Guyyaa Gaafa Kibxataa

Akkeenyaa bara baraa maqaan kee kan hin jijjiiramnef kan hin dabareedha. Kan hin rafneef kan hin mugne jabaa uumaa dadhabinni irra hin jiree, maqaa keef galata akka dhiyeessaniif kan uumte, samiif lafa irra kan jiry hunduu ija laphee isaatii gara keetti millisu (ilaalu). Hunduumaa kan dandeessuu ati Waaqa yaa Gooftaa.

Yaa Abbaa keenyaa jedhaa kadhadhaa.

Gooftaan gammachuu, uumaan sidha’o. Galatni baayyeen kan itti dubbatamu gara mana qulqullummaa kee dhufanii kan dadhabann, tajaajiltoota kee kan ta’an kadhannaa dubroofi dhiiraa simadhu.

Sibiraa kan argamu cuuphaa jireenya fooniif jireenya lubbuu kadhachuudhaaf kan dhufan cubbuu isaanii dhiisiif, hojii cubbuu isaaniis hin millatin, hin yaadinis qalbii jijjiirannaa isaanii simadhu. Akka arsaa bareedee jallatame, kadhannaa isaanis dhaga’i. Yaa uumaa keenyaa hundumaa ni argita ati garuu hin mul’utu. Abbaa dhiifamaa kan taatu yaa Gooftaa ija dhiifamaan nu milladhu.

Dheekkamsaa gara dhiifamaattis nu deebi’i. Addunyaa kana irrattis haa ta’u, addunyaa isa dhufu irraatti tokkichummaan akka si sagadnuuf, akka sii bitamnuufis tokkichummaa araarsummaa tolaa ilma kee kiristoosiin taanee maqaa kee akka ol ol kaasinuuf (Kabajinuuf) jabaannees akka si kabajinuuf har’aas yoomis ulfinnifi galatni jabinni isaaf Hayyaana qulqulluu wajjin kan jirti, hanga bara baraatti dhugummaan ameen!
