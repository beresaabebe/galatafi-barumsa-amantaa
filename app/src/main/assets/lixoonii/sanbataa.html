Galata (lixoonii) Guyyaa Gaafa Sanbata Duraa

Kennaa (arjummaa) badhasa jalqaba yaa Iyyesuus Kiristoos Abbaa isaaf humnaaf ogummaa isaati. Cubbuu addunyaa kan fageesitu hoolaan waaqayyoo sidha’o dhiifama nuu taasisi. Nuu gaddi, abbaan kee madalee ogummaa isaa isa laatuun samii irraa gadi buuteetta’o kan du’uuf kan jijjiiramu kan ta’e ati kan uume du’aadhaaf jijjiiramaa kan ta’ee saba kee akka fayyistuuf kochoo qillensaan kan deeman ergamoota ibiddaaf qillensa irraa kan uume sidha. Addunyaa qabdee kan jirtu hundumaa kan dandeessu ati yaa uuma, surraa (kennaa) kee afuura hurtuun lammii kee jabeessi jedhaa kadhadha.

Yaa Abbaa keenyaa jedhaa kadhadhaa.

Dhugumadhan yaa Gooftaa uuma, surraa kenna kee afuura hirtuun lammii kee jabeessi. Kan namatti ulfaattuf kan dhiphate kan salphistu sidha. Bareedinni duuka buu’otaaf horiin dadhabdootaa, namoota abdii kutateef abdiin isaanii sidha. Warra du’an kan kaastu yaa Iyyesuus Kiristoos humnaaf oguummaa abbaa kan taatu siin ni galateeffanna. Har’aas yeroo hundumaa hanga bara baraatti ameen!
