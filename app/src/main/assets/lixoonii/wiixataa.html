Galata (Lixonii) Guyyaa Gaafa Wiixataa

Hundumaa kan danda’u, hundumaa kan bitu Abbaa Gooftaa keenyaafi qoricha keenya Iyyesuus Kiristoos Waaqayyoon daballee ni waawwanna.

Halkan gutuu nu tikissee duukanaatii gara ifaatti nu baaserra’o, du’aa gara jireenyaatti, doofummaa (beekumsa dhabiisaa) gara beekumsa jabaatti nu geesse.

Halkan akka nu tiksee, jaalala jabaan, tokkichumaan akka jiraannu akka nu taasisuuf, humna ergamootaanis saba isaa akka eeguuf egi jenne si kabajinna. Eebba fooniif eebba lubbuu laachuuf taayitaa kan qabu. Waaqayyo Uumaa keenyadha, kadhadhaa.

Yaa Abbaa keenyaa jedhaa Kadhadhaaa.

Hundumaa kan dandeessu yaa Waaqayyo uuma, halkan akka nu tiksite, jaalala jabaa tokkichummaadhaan akka jiraannu nu taasisi. Ati halkan guutuu nu tiksiteetta’o. Yaa Gooftaa hundumaa kan dandeessu ati gara Gannataatti kan nu gaggeessu Ergamaa tolaa naa ergi.

Baayyina tolummaa keetiin dhiifama nuu kenni. Akka baayyina dhiifama keetiittis cubbuu keenya nurraa haqi. Lammii keenya keessaa nama tokko illee filatamuu irraa adda hin baasin. Maqaan kee nurratti waan waamamteef cubbuu keenya nurraa haqi, nuf dhiisin. Siif ilma kee Kiristoos fuulduratti jaallatamummaa akka argannuuf, argachummaa isaa nu kenni.

Isaa wajjin hayyaana qulqulluu wajjin ulfinaafi galata jabaa kan qabdu. addunyaan otoo hin uumamiin kaasee hanga harraatti, harras hanga dhufa Gooftaatti isaan boodaas hanga bara baraatti ameen dhugummaan.
