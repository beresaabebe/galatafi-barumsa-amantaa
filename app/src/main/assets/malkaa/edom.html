1. Yaa Giiftii Maariyaam, giiftii nagaadhahoo, nagaan siif haa ta’u jechaa, ganamaafi galagala galata harka fuudhii siifan dhiyiyeessa. Yaa Giiftii dubroo godoo giddu galeessa ganataatti dhaabamtedhahoo. Tajaajilaa kee kanan ta’e cubbuun koo, urjii samii caalaa kan baayyate waan ta’eef Haadha Iyyesuus Kiristoos hogganaa lubootaa dhahoo qalbii jijjiirannaa barbaachisu naaf kenni.
  2. Yaa Giiftii Maariyaam, akka aannaniikan garaa jiisuuf laphee kan qabbaneessu waamicha maqaa annan gaarii keef nagaan nita’af jechaa, nagaa nan jedha. Yaa Giiftii dubro, yaa gugee ko, yaa gugee koo jechaa kan si galateeffatu, gugee ogeessa solomoniitihoo, wallaalumaafi beelli akka nutti hin jabaanneef, nyaata ogummaa na nyaachisi. Hojii gaarii baayyee caalaa kennuun (gargaaruun) kan fooyya’eedhahoo.
  3. Yaa Giiftii Maariyaam, cubbamooni akka badaniif kan hin haawwine, kulkiif haaloo kan hin beekinne, lapheen kee gonkumaa garraamiidhahoo. Yaa Giiftii dubroo, kanaafis dhiifamni amala keetihoo naaf dhiisi. Lahpee koo keessa kan jiru kamiifi amamsaan siti himaa? gadda koof dhiiphina koo hundaa ati ni beektahoo.
  4. Yaa Giiftii Maariyaa, garee raajotaa, dhugomtotaafi wareegamtootaa keessaas yoo ta’e, gonkumaa akka kee kan an jaalladhu hin jiru. Yaa Giiftii dubroo, siin da’oo waanan taasifadheef, sitti amanatamuudhaan kan hafe hojiin koo hundi cubbuu qofa ta’eerrahoo, garagaarummaa kootiif dafii naaf ga’i.
  5. Yaa Giiftii Maariyaam, daandii ani irra deemurratti, bakka ani bulu hundaatti, guyyaas halkanis eegumsi kee narraa adda hin ba’in. Yaa Giiftii dubroo lubbuu lubbuuwwan hundaa kan nyaachistudhahoo, rakkina uffataafi nyaataa bara jireenya koo hundaa natti hin agarsiisin balaafi qorri kadhannaa nama ni irraanfaachiisutihoo.
  6. Yaa Giiftii Maariyaam, ammas dandii ani deemurra deemi, achi bakka ani buluttis tolummaan kee, tiksee naaf haa ta’u. Yaa Giiftii dubroo garee waan fedheenis Hasawaa koo naf bareechi, mirga harka keetii mataa koorra taasisi. Kan na morman mormii, gorsa hamaasaanis tokko tokkoon bittimsi.
  7. Yaa Giiftii Maariyaam, cubbuu hundaan xuraa’uurraa bishaan qalbii jijjiiraanaatiin qulqulluu akka nu taasisuuf, gaarummaan gara kootti dhiyaadhu. Tokkoon tokkoon keenyas maqaa silaaseen nu eebbisi.

    Yaa Giiftii dubroo, cubbamoota warra akka kootii kadhannaa keetiin gara qalbii jijjiirannaatti akka deebi’anif ogummaa afuuraa ibsiif. Dhugoomtotas hojiisaaniif eebba isaaniin, jabaatanii akka turaniif tiksiin.

  8. Yaa Giiftii Maariyaam, Kadhannaa koo simattee, gara samii gubbaatti akka ol naa baastuuf, hundara samii gararaa irra defii gara koo kottu.

    Yaa Gooftaa dubro, yeroo hundumaa eebba garraamummaa keetii, narratti firfirsi, badhadhinni biyya lafaa garuu, kan baduufi kan darbu waan ta’eef, tolummaa keetiin, dhugoomina akka naa buustuuf (kennituuf) barbaachisaa dhahoo.

  9. Yaa Giiftii Maariyaam Ufinni Addaam, simboon surraasaatiis, warqeen mataa henook sidhahoo. Yaaa Giiftii dubroo abishaalaafi beeka kan ta’e ijoolummaa isaatii jalqabee siin ni jaallata, dadhabaa qalbii dhabiisi garuu sababa tokko malee si jibba. Haa ta’u iyyuu malee, kiyyoo maree gorsa ammeenya isaatiitti inumti mataan isaa itti haa qabamu.
  10. Yaa Giiftii Maariyaam, sanyiin kan irratti hin kufinne qonnaan bulaanis kan itti hin dhiphaane, biqiltuu qurbaana qulqulluu kan horatte yaa garaamii, tattaaffattoota ergamootaa wajjin dafii gara koo kottu.

    Yaa Giiftii dubroo, addunyaan kan ishee jaallatu, deebiste ni balleessiti. Kiyyoo isheettis galshite, saba baay’ee summii isheetiin akka ajjeesite, hojii ishee kana kan irratti beeku, akkam nama beekaafi abishaala haa ta’uu? Kanaafuu, kiyyoo isheetiin akka nan qabinneef kiyyoo ishee kukkutii gati.

  11. Yaa Giiftii Maariyaam, jaamotaafi ifni isaanii, kan dheebotaniifis yaa burqa wayinii isaanii dafii gara koo kottu.

    Yaa Giiftii durboo, O wayyoo na dhiifta ta’aa? O wayyoo na tuffattaala? egaa gadda laphee kootii eenyuuttan himadha laata?

  12. Yaa Giiftii Maariyaam, gubbiin ibiddaa waaqummaa isaatii kan sin gubne, bultoo Waaqayyootihoo dafii gara koo kottu. Yaa Giiftii durboo bara hundaa yeroo hundumaaas na fayyisi, dhiifamas naaf taasisi, Ati yoo na fayyiste, dhiifamas yoo naa taasiste kan natti murteessu eenyuudhaa, lubbuuttis foonittis kan murteessu Ilma keetihoo.
  13. Yaa Giiftii Maariyaam haadha mootii samiifi lafaatihoo, cubbuu koo hunda gonkumaa balleessuuf daftee gara koo akka dhuftuuf jecha sin waamadha.

    Yaa Giiftiii dubroo, rakkina lubbuuf foonii hunda narraa ittisi. Keesumattuu asiif achi jechuudhaan guyyichi akkasumaan akka natti hin dhiineef, hojii bu’aa qabu irratti akka ani argamuuf jecha, ani abba badii tajaajilaan kee sin waamadha.

  14. Yaa Giiftii Maariyaam, lakkoofsa ga’aa kan hin qabne, barbaacha addunyaa kana hordofuudhaan, ani cubbamaa dadhabaa tajaajilaan kee sin gaddisiisata’innaa.

    Yaa Giiftii dubroo, Egaa durii jalqabee, inaaffafi haloo dhiisuun amala keetihoo dhiifama naa taasisi yaa haadha abbaa dhiifamaa naa taasgabbeessi malee nan dhiisin.

  15. Ejjaafi hojii badii hojjachuun, soba dubbachuun sin gaddisiise ta’innaa. Yaa dubroo haa ta’u malee siin laphee guutuun jaalachuun salphaa natti hin fakkaatu, lubbuu torbaatamii saddeeti kan nyaate, Balaayesab bishaan barruu harkaatiin gannata seenuuf ga’eerahoo.
  16. Yaa Giiftii gara laafetti, xura’inniif mudaan otoo sirra hin jiraatin dubrummaan jabaattee kan jirtu, jaalalli kee ibidda ta’ee marrummaan na gubahoo. Yaa Giiftii dubroo yeroo hundumaa waan gaggaarii naaf taasisi. Akka hojii kootiitti du’atu naaf ta’a ture. Haa ta’u malee, akka ani qalbii jijjiiradhuuf yeroo yartuu naa obsi.
  17. Isin biyya lafaa kana utuu jiraattanii, gadiifi gubbaa kan buutaniif kan baatan hundi keessan, jaalalli giiftii keenyaa salphaa isinitti hin fakkaatin. Yeroo hundumaa maqaa ishee waamaa jaalala giiftii keenyaaatti kan amanatame gaadidduu waawwannaa ishee jalatti kan gaadiddeefame, ejjaas yookiin amala xuraa’aas kan qabu yoo ta’e iyyuu gonkumaa dubroo ni ta’a.
  18. Yaa Giiftii Maariyaam, bultoo Waaqayyoo waan taateef, baayyee kan ta’ee raajii kee adda addaa gurra kootiin dhaga’eera, ija koonis argeera. Yaa Giiftii dubroo ana jooraafi jawwaraa kanaan ta’e addunyaa kana irratti kan na tasgabbeessuuf na sossobu dhabeerahoo. Yaa haadha gara laafetti mee naatti siqii jecha sossobinaa na dhageesisi.
  19. Yaa Giiftii Maariyaam, Raawwata kan hin qabne raajii kee baay’ee kan ta’e fakkeenya urriin addunyaa iraatti ni bittinna’e.
    yaa Giiftii dubroo, godoon Phaawuloos nama roomaati, anis utuu qalbii hin jijjiirratin akkan hin duuneef, umurii akka naa dabaltuuf solomeen kan baatte Ilma keen sin kadhadha.
  20. Yaa Giiftii Mariyaam, gadda irraa kan na tasgabbeessitu, dulluma koo dargaggumaan kan naa jijjiirtu, barri kan hin lakkaa’amneef daa’imni garaa kee keessa buleerahoo. Yaa Giiftii dubroo abdii kan kutataniif abdii, isaaniitihoo, kadhannaafi ixaana akkasumas yeroo qurbaana qulqulluu dhiyeessan ana tajaajilaa kee tasgabbeessuu dhaaf bakka kanatti argami.
  21. Yaa Giiftii Maariyaam, maqaan kee cubbamoota ni qulqulleessa, dadhabootas, aishaalaafi tattaafataa ni taasisa. Yaa Giiftii dubroo ani yaadaan (sammuun) iyyeessa kanan ta’e tajaajilaan kee qofaa afeeraatii, gadda gara kootii sitti himee akka na tasgabbeesituuf jecha, dafii gara koo kottu.
  22. Yaa Giiftii Maariyaam, rakkinni lubbuufi foonii akka narra hin geenyeef, maqaa kee isa meetii, galmee ulfinaa irra yookiin saanduqa keessaa fudhadhee gatii nan taasifadhe. Yaa Giiftii Dubroo (bareedina) ifa fuula keetii ilaaluu nan barbaadahoo, yaa dubroo, dafii naaf ga’i. Yaa dubroo deemii naaf ga’i, qaamni koo si dheebotteettihoo.
  23. Yaa Giiftii Maariyaam, guyyaa gaafa murteen dhumaa kennammutti hundumaa gara daandii jireenyaatti kan harkistu urgeenyi kee kan minyaa’u, urgeenya amantiiti hoo.

    Yaa Giiftii dubroo dhugoomtonni, hojii gaarii isaaniin, du’a isaaniin kan si argatan mootummaa samiitihoo, urgeenyi jaalalla keetii akka urgeenya ixaana adii natti urgaa’e. Jaalala wayinii kee mi’aa’uunis lapheen koo ni gammada.

  24. Yaa Giiftii Maariyaam, qulqulloonni abbootiin keenya sii filatan, qoricha isaanii kan taatu siinis, ija gaarii ni argamsiisan (horan). Yaa Giiftii dubroo dadhabaadhaaf tullu ga’oo isaatiiti. Dugoomaan hojii isaan ni fayya, cubbamaan akka kootii garuu itti baqiin isaa, da’oon isaa sirraa kan hafe gara kamidha.
  25. Yaa Giiftii Maariyaam kitaabonni akka dubbatan yookiin akka ibsanitti, hojii mullata Waaqayyoo kan fakkaatu, bifa keef nagaan ni ta’a.
    Yaa Giiftii dubroo qabeen urgooftuu saamu’eel sidha hoo. Kan si jaallataan, kan si kabajan hundi eebba kee haa argataan, kan si jibbaniif kan sitti inaaffan garuu dafanii haa badan.
  26. Yaa Giiftii Maariyaam, misirroo abbaa siif nagaan ni ta’a. Yaa haadha ilmaa (Iyyesuus) siif nagaan ni ta’a. Buultoo afuura qulqulluu yaa Maariyaamgalmee qulqulleetti nagaan siif ni ta’a. Warqeef meetiin kan baduuf kan darbuudha. Kanaafis badhaadhina addunyaa kana caalaa sii jaallachuu nan filadha hoo tolummaa kee naraa hin fageessin.
  27. Yaa Giiftii Maariyaam, ammas ganamaafi halkaan na tiksii, rakkina fooniif rakkina lubbuu irraa na baraari bara baraan Ameen!
