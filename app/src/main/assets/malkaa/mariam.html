1. Yaa Giiftii Maariyaam, baay’ee kan miidhagdu waamicha maqaa keetiif nagaan haa ta’u. Kalbanii, qusxiisan balax urgoftuu jedhaman irra urga’ina jaallatamu qabahoo. Yaa dubroo ulfina guddinaa sillasee biraa kan argatteedha. Biqiltuun gaarii bishaan jallisiin akka lalisu, jaallachu wayiniin lubbuu koo qulqullummaa dheebochu irra yeroo hundumaa haa quufu.
  2. Yaa Giiftii Maariyaam, gadi bu’eenyi isaa tulluu gali’adi irraa kora ra’ee bu’u kan fakkaatu rifeensa mataa keetiif galanni haa ta’u. Jalqaba akka afaan abbaa keetiin dubbata miti. Yaa dubroo ati, galimmaa qulqulluu kan ta’e mana bultii waaqayyooti. Dinoonni koo, kan inni itti ofi jajju giiftiin isaa essa jirti jedhanii akka natti hin qoosneef, ana garbicha kee gaddaaf dhiphinatti dabarsitee nan laatin.
  3. Yaa Giiftii Maariyaam, dibaa addunyaa biyya kanaa kan hin taane, urgoftuu galata qabu dibachuu kan ta’eefiif mataa keef nagaan haa ta’uuf.
    Yaa dubroo lubbuufi foonin, keessaafi alaan ati qulqulluudha. Ana garbicha cubbamaa kee, kennaa qurbana barbadaa isaa isa qulqulluudhaan akka na qulqulleesuuf luba samii suraafii naa ajaji.
  4. Yaa Giiftii Maariyaam, ifi Waaqumaa gonkumaa kan tokko ta’e ifa fuula keetiif nagaan haa ta’uuf, aduu, urjiifi ji’arra baay’ee kan calaqisuudahoo.Yaa dubroo, karaa ba’a koof gala koo naa qulqulleessi. Bariif halkan dhokatee kan saggoo namaa rukutu irraa na tiksi. Inni (Seexanni) badii kiyyoo isaatii qopheesee boollaas qotee na eegahoo.
  5. Yaa Giiftii Maariyaam, adda kan ta’an ija keef nagaan haa ta’u. Inaaffaan Qayyaaffaf Haannnaan yemmuu ilma kee dhiphise, hurri imimmaan qabate ta’eerahoo.
    Yaa dubroo, qulqullummaan kan guutamteedha. Abbaa oduu gammachiisaa, qulqullaa’an Gabri’eel gammachuu isaatiin akka si gammachiise, dubbii laphee koo gammachiisu na dhageessis.
  6. Yaa Giiftii Maariyaam, akka bishaan ji’a onkololeessaa kan qulqullaa’aniif akka urjii barii kan ifan ija keef nagaan haa ta’u. Iyyummaan kan guddataniif rakkatan ni ilaaluhoo.
    Yaa dubroo, lapheen shakkiif sammuun xura’aa kan sirra hin jirre qulqulleetiidha. Finyoo bo’ichaa jabaan kanan marifatame akka ani hin taaneef, lapheen koo yemmuu gadduuf yemmuu yaaddoon galu akka na jajjabeesituuf dafii gara koo kottu.
  7. Yaa Giiftii Maariyaam qulqulluu kan ta’an gurroota keef nagaan haa ta’u. Gooftaa hundumaa kan ta’e Waaqayyootti fakkeefamee kan dhufe jecha Gabri’eel dhaga’uuf ga’anii argamaniiruhoo.
    Yaa dubroo, dhiifamaaf tolumaan kan jabaatte waan taateef, lapheen koo ibidda guddaan yemmuu gubatu hiraanfatummaan kan irra hin jirre gurri kee dubbii koo haa dhageefatu.
  8. Yaa Giiftii Maariyaam, ijoollonni Raaheeliif Liyaa yemmuu Ilma kee fannisan, imimmmaan ibiddaan kan gubataman maddii keef nagaan haa ta’u.
    Yaa dubroo baay’ee kan amanameef kan ajaa;ibsiifame ogummaa keetiin, lammii gumaa kan ta’u leenca bosonaa (seexana) irraa naa fayyisi. Abdiin hirkoo lubbuu kootii sidha’oo.
  9. Yaa Giiftii Maariyaam, hurgeenya gaarii kan qabdu, burqaa hurgooftuu kan ta’an funyaan keef nagaan haa ta’u. Ulfaataa kan ta’e urgeenya ixaana irraa adda kan ta’eedhahoo.
    Yaa dubroo, Yaa’iqoob fakkenya warqee riqaatiin akka si arge, ati anaafis nooraa itti ce’umsa jireenya kootiif abdii koodha, kanaafu gaafa murtii wabii akka naa taatuuf sin kadhadha.
  10. Yaa Giiftii Maariyaam, daraaraa diimaa kan fakkaatan hidhii keetiif nagaan haa ta’u. Qulqullummaan kan bareedani dhahoo.
    Yaa dubroo, baha ifa dhugaa kan taate, balballi (karri) bahaa siidha. Gaddaaf bo’icha irraa gonkumaa na fayyisi, gaddi laphee kootii baay’eedhahoo.
  11. Yaa Giiftii Maariyaam, abbaa jaarsolii yemmuu ta’u, sirraa nama ta’ee yemmuu ijoollee ture, barbaadaa waaqummaa, isaa kan dhungate afaan keef nagaan haa ta’u.
    Yaa dubro, akka ixaana qulqulluu, hurgeenya kan qabdudha. Fuulli koo badii qaanii, yemmuu haa guuggattu, yaa haadha koo badii koo narraa balleesitee akka na tasgabbeesituuf ariifadhuu naaf ga’i.
  12. Yaa Giiftii Maariyaam, fakkeenyi yoo dhibeefiyyuu aannan sa’aa kan fakkaatan harkaa keetiif nagaan haa ta’u. Erga quncifamanii booda miicamanii akka hoomaa qulqulla’anii qulqulluudha’hoo.
    Yaa dubroo, dabteraa oriitiif ati ragaa seera isheetiiti, Ilmi kee dhiiga qorsiisee lubbuu itti gochuudhaan kan na uume anaan jabeessi na barbaadi, surraa kana caalu hin barbaaduhoo.
  13. Yaa Giiftii Maariyaam, afaan kee isa burqaa dammaafi aannanii ta’eef nagaan haa ta’u. Kunis argama jaalala tokkumaati.
    Yaa dubro, Iyyaaqemiif qulqulleetiidha. Ana garbicha kee ija diinaa irraa na olchi, xuraa’umaa irraas qulqulluu na taasisi. Samiif lafa irratti abdiin koo si’umadhahoo.
  14. Yaa Giiftii Maariyaam, jecha kee isa wayinii caalaa minyaa’uuf nagaan haa ta’u. Dhukkuba kan namarraa faccisu sagalee baganaa caalaa kan miidhagudhahoo.
    Yaa dubroo, baataan (teesson) ifa jireenyaa sidha. Jaalalli biyya lafaa buubbee isaa yommuu natti kakaasu yaa haadha dhiifamaa akka narraa callisuuf ifadhu.
  15. Yaa Giiftii Maariyaam, urgeenyi isaa jireenya kan namaa haaressu hafuura keef nagaan haa ta’u. Gennata keessatti kan argamu akka muka urgooftuu filatamtee, urgeenya adda ta’e qabahoo.
    Yaa dubroo, qulqulloota kan qulqulleessuuf, mana bultii isaa ni taate. Yaa gugee gaarii daandii gaariirra na gaggeessi, du’a irraas na olchi, gaadidduu eebba kochoo kee jalas na dhoksi.
  16. Yaa Giiftii Maariyaam, dursee akka solomon dubbatetti jechi wayinii caalaa minyaa’u kan keessaa burqu laagaa (qoonqoo) keef nagaan haa ta’u.
    Yaa dubro, Ati uffannaa ifa guddaa kan aguuggatedha, kanaa fiyyuu qulqullummaa irraa qullaa ta’uun akka nan dhiphisinneef, ogeesonni kan itti hin dhiphanne, warri habshaalonnis kan itti hin dhibaane uffannaa surraa naa uffachiisi.
  17. Yaa Giiftii Maariyaam, akka dubroota warra kan biroo meetiif warqeedhaan kan hin taane, habilii qulqullummaa kan diratte mormaa keetiif nagaan haa ta’u.
    Yaa dubro, gugee bifti ishee adii taatuun si fakkeesina, barri jireenya kootii dadhabinaan darbuuf yeroo dhiyaatu qalbii jijjiiradhee waan gaarii akka ani hojjadhuuf umurii koo naaf dabali.
  18. Yaa Giiftii Maariyaam, kiruubeeli kan baatan Waaqayyoon, saggoo kee waarra baataniif nagaan haa ta’u. Waanjoo badii irraa kan fagaatannidhahoo.
    Yaa dubroo, bareedinni badhaasa Hoose’i sidha. Yaa giiftii koo, ilmaa keen akkana jedhi, lapheen koo kan sitti dhiphateef, hiika siif haa barbadu, akka ani isa galateefadhuufis daandii qajeelaarra si haa gaggeessu nan jedhi.
  19. Yaa Giiftii Maariyaam, Kiruubira kabajamaa kan ta’ee duddaa keetiifi nagaan haa ta’u. Ibidda baatee, boba’inni ibiddummaa isaatii hin gubbinehoo.
    Yaa dubroo, otoo hin turin dafii naa ga’ii rakkoo koo hundumaa narraa fageessi. Yaa Giiftii koo hawwii koo isa gaggaariis naa raawadhu.
  20. Yaa Giiftii Maariyaam, ibidda waaqummaaf boqonnaa kan ta’ee laphee keef nagaan haa ta’u. Kunis qaama kan qulqulleessudha ho’inni isaas lubbuu kan fayyisudha.
    Yaa dubroo, rakkatootaaf qabeenyaan warqeesaanii sidha. Biyya fagoo deemee waan uffatu dhabuudhaan yoo qullaa hafes ati uffannaa isaaf galaa isaa waan taateef hin gaddu, ati jirtaafihoo.
  21. Yaa Giiftii Maariyaam, ogummaa samiirraaf boqonnaa kan taate garaa keetiif (gadameessa keetiif) nagaan haa ta’u. Kan dorgomuuf kan fakkaatu hin jiruhoo.
    Yaa dubroo jaalala abdiin guutamtee laachuuf mirga qabda. Yaa Giiftii koo, Simbirroonni hucoolee ishee kochoo isheetiin akka walitti qabdu, anaanis fuula diina koo duratti abdii naa ta’ii na dhoksi.
  22. Yaa Giiftii Maariyaam, ibidda Waaqummaatti kan bu’an harkoota keef nagaan haa ta’u. Isaanis albuun birriif qubeelaa warqeen akka miidhagan kan hin feenedha.
    Yaa dubroo, iji gadameessa masentootaa sidha. Yaa Giiftii koo, cubbuu addunyaa kana irraa qulqulluu naa taasisi. Qaama koo irraatti mootii taatee akka jiraattuuf hin fedhuhoo.
  23. Yaa Giiftii Maariyaam, humna kan argatan saboota keef nagaan haa ta’u. Ibidda Waaqummaa baachuudhaaf dandettii qabuhoo.
    Yaa dubroo, ati fakkeenyi kan siif hin argamne qulqulleettiidha. ani cubbuun garbicha keetii yemmuu baayyatu akka gamoo baabiloon jijjissi.
  24. Yaa Giiftii Maariyaam, galateeffachuun koo baay’ee barbaachisaadha. Ibidda waaqummaa isa gubbuuf da’o taateettahoo.
    Yaa dubro, ankisoo kan jedhamu hurgooftuu beekamaarra kan filatamu urgeenya qabiddahoo. Yaada laphee kootii yemmuu hundumaa akka ani sitti himuuf, otoo narraa hin fagaatin yeroo hundumaa na wajjin jiraadhu.
  25. Yaa Giiftii Maariyaam, humna Waaqayyoon kan jabaatan ciqilee keetiif nagaan haa ta’u. Naazireetiif galilaati, warqii jiribii wajjin wal taasisee yemmuu fo’an dadhabbiin itti hin dhagaa’aminne’hoo.
    Yaa dubro, Yoodiitiif burqaa dubbii isheetiif maddi ogummaa isheetii sidha. Gammachuuf itti tolummaan bakka inni jiru akka na jiraachistuuf dafii saragalaan gara koo kottuuti waadaa naa seeni.
  26. Yaa Giiftii Maariyaam, magariisaaf cuculuquun kan irra adda hin baane barru keetiif nagaan haa ta’u. Buddeen fayyinafi dhugaati qorichaa qoratame laachudhaaf dafanii diririfamuhoo.
    Yaa dubroo, yeroo hundumaaf abiddii si gadhachudhaan sin wammadha. Lammii lammiidhaan (kutaa kutaadhaan) galata kee akkan dubbadhuuf ana dadhabaa garbichaa keetiif laphee qalbii qabu naa ladhu.
  27. Yaa Giiftii Maariyaam, lakkofsaa irraa osoo hin hir’atin, xuraa’inis osoo irra hin jiraatin, giddu galeessa harka keetii irra kan biqilan (argaman) quboota keef nagaan haa ta’u.
    Yaa dubroo, fageessee kan inni ittiin waamu tulluun raajicha Eliyaas sidha. Ilma dhala namaafi tolaa irra waan badii kan yaadu bineensa hamaa irra humna tolummaa keetiin naa eegi.
  28. Yaa Giiftii Maariyaam guyyaa gara guyyaatti walumaa gala, xinnoo xinnoon kan guddatan, qarqaara quba harka ketiitti kan biqilan qeensota keetiif nagaan haa ta’u.
    Yaa dubro, ifa duukkana kootii waan taateef, ana garbicha kee giddu galeessa umurii kootiitti nan fudhatin, dhiphinni laphee kootii kadhahoo.
  29. Yaa Giiftii Maariyaam, xuriifi mudaan kan irra hin jirree harmaa kee lamaanuuf nagaan haa ta’u. Annan dubrummmaa afaan waaqayyoo keessatti maddisisteettahoo.
    Yaa dubroo, nagaa kan argamsiiste giiftii nagaati. Iyyeessa kanan ta’e anaanis harma keetin na guddisisi malee (sirra kan caaltu) hadha kan biraa hin barbaaduhoo.
  30. Yaa Giiftii Maariyaam, Uffannaan ishee uffannaa warqeetti jedhe Daawit akka dubbate, Warqii sabbata qulqullumaa kan godhatan mudhii keef nagaan haa ta’u.
    Yaa dubroo, Yaa’iqoobiif ulfina surraa isaattiiti ana garbicha keen sin jaaladha’o, kanaaf iyyuu jaallataa kee ulaan taateef enyuutu sitti bu’a kan jedhu abdii naa laadhu.
  31. Yaa Giiftii Maariyaam, kennaa baay’ee kan ta’ee garaa keef nagaan haa ta’u. Seeri kakuu moofaa kan itti barreeffame tsillaatii museera kan caalu ulfina guddaa qabahoo.
    Yaa dubroo, humni kee nama badii kan ta’e diina koo akka narra balleesuuf, bo’icha baay’eedhaan faannisa hanga badutti irreen kee lafatti akka dabaluuf jecha, yeroo hundumaa siin waammadha, sin kadhadhas.
  32. Yaa Giiftii Maariyaam, dallansuuf, waa’ee badiif gatii deebisu irraa kan fagaatu laphee keef nagaan haa ta’u. Amanamtumaa keetiif aduun ba’aaf gala isheetihoo.
    Yaa dubroo, ijoolota warra ebbifamaniifi filatamaniif iddoo bashanana isaaniiti. Qulqullummaa irra qullaa kanan ta’e ana garbicha kee uffannaa misirummaa natti huwiisi. Warqee daraaran kan hojjatame uffannaa jaallatamaa haguuggannaa (Magonatsefiyaa) nati haguugi.
  33. Yaa Giiftii Maariyaam, Lamaan kalee keetiif nagaan jedha.Dibaa lubumaafi kan nama gammachiisu. Meeshota wayiniin ittin cuunfamudha.
    Yaa dubroo, diinota kee fulduratti ol ka’u hin sodaatin, anaaf jedhee ilmi koo jaallatamaan wallitti qaba giddutti gargaara siif ta’a nan jedhi
  34. Yaa Giiftii Maariyaam, waan gaarii yaaduun amala isa kan ta’ee samuu (yaada) keef nagaan haa ta’u. Dhiifama gochuun hojii isaa, yeroo hundumaatihoo.
    Yaa dubroo, akkenyaa Daawit irraa kan argamte ati dhaloota dhala Daawititi. Badii koo irra akkan qulqullaa’uuf fakkeenyi kan hin argamneefiif foo’aafi bishaan qulqulluu natti facaasi. Bishaan jireenyaatiinis na dhiqii cabii irraas kanaan qulqullaa’u naa taasisi.
  35. Yaa Giiftii Maariyaam, Ilmi kee qaraaniyootti fannifamee yeroo oole, gadda ibiddaan kan gubaate laphee keef galaanni haa ta’u.
    Yaa dubroo, ati golgaa cobaafi kan raajiiti cubbuu dhiisisuuf tufiin sirra hin jiruhoo. Akka seera lubaatti kadhannaa koo ol naaf baasi.
  36. Yaa Giiftii Maariyaam, dabtarraa Museen (godoo) kan fakkeesame qabeenya horii keessa keetiif nagaan haa ta’u. Isheen kunis humni jabaan kan gaadiddeesudha.
    Yaa dubroo ijoolota saaratiif oi jaajjoo isaaniiti.Mucaa Qisiraa, yaa Giiftii koo, dhiphinaafi dadhabina duukkana biyya lafaa kana irraatti na fayyisi. Dandiin ishee dhiphoodhaafi kan keessa daddaribanis jajalladha’o.
  37. Yaa Giiftii Maariyaam, Uffannaa mana qulqullumaan kan fakkeesamu qabeenyaa keessa keetiif nagaan haa ta’u. kunis Iyyesuus Kiristoosidha.
    Yaa dubroo, Bobaa’an ibiddaa kan hin gubinnee biqiltuu gamoojjii sinaan si fakkeessina. Laphee azimaarii sagaleen muuziqaa akka dadamaqsu, anaanis akkasuma sagaleen afuura keetii, lubbuuf fooniin naan haa dadamaqsu.
  38. Yaa Giiftii Maariyaam, Taa’umsisaa akka meetii guutichaa gurraa kan ta’e hannuurra keetiif galanni haa ta’u. Innis jireenyummaa isaa irraa kan qullaa hin hafnedha.
    Yaa dubroo osoo hin dhaabin kan biqiltee, bishaanis osoo hin obaasin kan laalistee, filimaata uleen Aroon sidha. Yemmuu ani kadhadhu, dubbii afaan kootiitti hir’inni yoo argamee xoofoo kiruubiin (kirrub) ati itti naa guuti.
  39. Yaa Giiftii Maariyaam, Bultii Waaqayyoo kan ta’e gadameessa keetiif nagaan haa ta’u. Harki mirgaa Waaqummaa isaatii kan si toltolichedhahoo.
    Yaa dubro, Urgooftuu naardoos jedhamu irraa baay’ee caalummaa kan qabu, urgeenya qabda. Yaa giiftii koo waan barbaade hundumaa argachuuf iji koo inni tokko, isa lammaffaa wajjin gara kee ni mil’atta.
  40. Yaa Giiftii Maariyaam, Mana qulqulleetii qulqullummaa Waaqayyoo kan ta’e, gadameessa keettiif nagaan haa ta’u. Seerriif heerri isaa warra kan biraarra adda kan ta’edhahoo.
    Yaa dubroo, wayinii jireenyaa damee margaa jedhamtee galatoomfamaa ni jiraata. Yaa Giiftii, cubbuun kee siif haqamteettii, kadhannaa kees ol si baaseera naan jedhi.
  41. Yaa Giiftii Maariyaam, durbummaa yaadaafi foonii keetiif nagaan haa ta’u. Isa keessaa aduu qulqullaa’a kan jedhamu Iyyesuus Kiristoos argameerahoo.
    Yaa dubroo, bareedini qulqullummaa keetii fuula waaqayyoo duratti kan jaallatameedha.Mucaa abbootti warra tolootaa sanaa yaa giiftii yemmuu ani karaa dheeraa (fagoo) deemu bakka ani bulu hundumaatti eegumsi kee narraa adda hin ba’in.
  42. Yaa Giiftii Maariyaam, ulee leewotaa Aaroon jedhamee kan waamamaan mudhii keef nagaan haa ta’u. Cobni osoo sin giddisin, burqituu osoo sin obaasin lalistee argamteettahoo.
    Yaa dubroo, namaafi ergamootaafi ati giiftii isaaniiti. Lammii kee filatamanii wajjiin akka ani jiraadhuuf jabaa eebbaa keetiin bohaartiif gammachuun bakka jirutti na geessi.
  43. Yaa Giiftii Maariyaam, Osoo adda hin bahin kan jiraatan jilboota keef nagaan haa ta’u. Uumaanni addunyaa biyyaa kanaa gonkumaa qoruu hin danda’anihoo.
    Yaa dubroo, cubbuun kan sirra hin jirre jabduudha. Yeroo hundumaa ganamaafi galgalaa tolummaan kee naaf haa ta’u.
  44. Yaa Giiftii Maariyaam, Waaqayyoof kennaa taasisanii yemmuu mana qulqullummaatti si galchan, jabaaf galata galchuuf kan jilbeenfate jilbaa keef galanni haa ta’u.
    Yaa dubroo ati akka teessoo dhagaa kabajamaati. Cubbamoonni waltajjii gamachuu guddaa irra yemmuu ari’ataman, booji’ama kee akkan ta’uuf isaan irra adda baasii naa hambisi.
  45. Yaa Giiftii Maariyaam, Kutaa biyya yihuudaa kan taate Beetaliheemiiti hanga Gibtsitti, fiigivhaan kan dadhaban miilota keef nagaan haa ta’u.
    Yaa dubro Boba’an ibiddaa kan hin gubnee muki araba ibiddaan bareede sidha. Fooqii kattaa irratti ijaarame jiraachuu akka ani danda’uuf beekumsaan ol ol na qabi, akka ani hin kufinnefis hirkoo naa ta’i.
  46. Yaa Giiftii Maariyaam, Koomee millaa keetiif galanni haa ta’u. Dandii irratti mucucinni hin mucuceesuhoo.
    Yaa dubroo, Nohiif mallatoon waada sabbata Waaqayyooti. Haadhi mucaa isheetiif akka gootu urgeenya minyaa’un na mi’eesi. fayyina naa laadhu.
  47. Yaa Giiftii koo kophee kee isa dhiphina baay’ee argeef nagaan haa ta’u. Sodaa loltoota Heroodis warra nama ajjeesan irra kan ka’e kophee malee qoraattifi bosona nama gubu keessa deemaniiruhoo.
    Yaa Tsiyoon, Qubni Waaqayyoo inni dhugaan (sagaleeabboommiin kurnanuu) kan sitti bareefame tsilaatiin muusee sidha. (Jabboonni)Gugeewanni ijoolota isaa nii caalaa siin akka si jaalatan, har’aa jalqabee sin jaalladha eebbi kee narraa adda hin ba’in.
  48. Yaa Giiftii Maariyaam, ja’afi afur walliti eda’amanii kudhani kan ta’an quba miillaa keetiif nagaan haa ta’u.
    Yaa dubro, teesso arsaa qulqulluu kan jedhaamtedha. Afaan koo maqaa kee yemmuu waamu fiigichaafi ariitiin naa ga’i.
  49. Yaa Giiftii Maariyaam, Afuura qulqulluun kan bareedan qeensaa millota keetiif nagaan haa ta;u. Asoonsillaa bulluqisanii ittin ofi bareechisuu hin feenehoo.
    Yaa dubroo, ati yaadannoo siidaa Iyyasuu ilma Naweeti. Bineensaa adamsaan dhiibbaa Diyaabiloosiin akka ani hin kufineef hirkoo naa ta’i. Duras hadha koo hewaan irratti dhiphina guddaa kan fide isa dhahoo.
  50. Yaa Giiftii Maariyaam, Akka muka Meexii isa madda bishaan qarqaraatti biqile kan fakkaatu qaama keef nagaan haa ta’u.
    Yaa dubroo, Kadhannaa koo akka naa dhageesuuf siin kadadha. gara gadiitti akka ani hin kufneef gara gararaatti ol na baasi. Namaa gadduuf akka kee kan badhaadhefame hin argadhuhoo.
  51. Yaa Giiftii Maariyaam, Akka aduu kan ifu bifa keef nagaan haa ta’u. Addeenyi ifa isaatii yeroo hundumaa hin hir’atuhoo.
    Yaa dubroo Haadha jaalalaa, nagaa kan jaallatudha. Jabaa kan ta’e ifaan kan guutame fuulli kee natti haa mul’atu. Eenyuu iyyuu akka inni dhaga’uu kan ani hin barbaanne dubbii siyaadachisun qabahoo.
  52. Yaa Giiftii Maariyaam, Dhiphina malee foon ishe irra adda kan baate lubbuu keetiif nagaan haa ta’u. Jireenyi ishees ulfinaan mirga ilma keetiiti.
    Yaa dubroo, fuula uumaa duratti surraa ulfinaa kan aguugattedha. Eebbaa keetin qaama koo dadhabiii irra boqochiisi ulfinni ishees ulfina toloota abbootishee gadi hin ta’in.
  53. Yaa Giiftii Maariyaam, akka inquu (dhagaa kabajamaa) foon kee kabajameef nagaan haa ta’u. Warqeef birrii calaa adda kan ta’ee Fakkenyaa qulqullumaa qabahoo.
    Yaa dubroo, ati iddoo biqiltuu wayinii kootiiti. Kabajamaa maqaa keetiif galata ani dhiyyeesse lagatti akka hin afineef gara samiitti ol naa baasi.
  54. Yaa Giiftii Maariyaam, Gatiin isaa baay’ee mi’aa kan ta’eef urgeenyi isa baay’ee kan minyaa’u, harka duuka bu’oota Gooftaatin kan kafanaame foon kee isa qulqulluuf nagaan haa ta’u.
    Yaa dubroo, lammii warra eebbifamaniifi filatamaniif mucaa isaanii ishee eebbifamtuudha. Yaa Giiftii koo tolummaan kee akka harmaa ta’ee akka na guddisuuf sin haadhafi abbaa koo taasifadheera.
  55. Yaa Giiftii Maariyaam, Geteesamaanii kan jedhamu bakka awalcha keetiif nagaan haa ta’u. Isheenis Iyyerusaalemiif dhiyoo kan taatedha.
    Yaa dubroo, Gugee mootummaa Waaqayyo jedjamtee ni galatomfamita, foon kee iddoo itti boqotee kan taate geetee samaaniinis, saabaa irra biyyoo urgooftuu, biyyoo irra immoo siidaan waarqee ni dhiyaataaf.
  56. Yaa Giiftii Maariyaami, Ammas goodina Iyyerusaalem kan taatetti awwaalisha kee isa argamuuf nagaan haa ta’u. Kunis maqaan isee Geetesamaanii jedhamti.
    Yaa dubroo, aafuraan kan dhiphataaniif yaadaan kan dadhaban ati guyyaa boqonnaa isaaniiti. Ifa qulqullotaa kan ta’e aduu qulqulluun Iyyesuus Kristoos sirra yemmuu bule, humni jabaan surraan si golgee.
  57. Yaa Giiftii Maariyaam, Iddo fagoo gammachuun jirutti kan olba’e foon keef nagaan haa ta’u. Kunis qoratamuu kan hin danda’amne Inquudhaan kan tolfametu keessatti argama.
    Yaa dubroo, kan daaraafi kan hucuu kan hin taane, ati ifa godooti, bookkaan gannaa qalama ta’udhaan samiinis waraqaa yoo ta’ee galatni kee barreefamee kan dhumu miti,
  58. Yaa Giiftii Maariyaam, Ammas yaa mucaa mootii yihuuda, lafaa gara samiitti kan olba’e foon keetiif nagaan haa ta’u. Badhadhuma obboleessa isaa kan leewwii kan taate siif galanni haa ta’u.
    Yaa fubroo, urgeenyi kee urgeenya daraaraa abaaboo ni caala, ballini lafaa akka gabateetti osoo qopha’ee bultitti gara bultitti haaraafi keesummaa kan ta’u galata kee umaminni bareessee fiixuu hin danda’u.
  59. Yaa Giiftii Maariyaam, Iddo jannataa karaa bahaa gidduuti, bakka ifi godoota itti dhaabbatetti kan ol ba’e foon keef nagaan haa ta’u.
    Yaa dubroo, afaan hundumaan galatoomifamuun kan sii ta’u waan ta’eef, samii irra kan jiran ergamootaaf lafa irra kan jiraatan ijoollee Addaam biraa yeroo hundumaa galanni sii dhiyyaata.
  60. Yaa Giiftii Maariyaam, Sagalee galata kanaan, nama si kadhatuuf, yeroo dhiphinni irra ga’u, dafii ga’iifiitii jabaadhu sii jira’o ittiin jedhi.
    Yaa dubroo, Ati dhagaa kabajamaa kiristiloobee jedhamudha. Hojiin kees urgeenya qumbii irra adda kan ta’edha. Gadameesi kees urgaa’u argamsiiseerahoo.
  61. Yaa Giiftii Maariyaam, Amma lakkoofsa rifeensa mataa kootiin siin galateefadha. Amma lakkoofsa lafee kootiinis sin galateefadha.
    Yaa dubro, amma lakkoofsa lafee kootiinis siin galaateefadha. Yaa mootatti, lakkofisa hin mul’anneefi itti hin bu’amnee hundaan nagaa siin jechaa. Galata durbummaa keetii afaan koo yeroo hundumaa ni dubata. Hanga bara baraati Ameen!
