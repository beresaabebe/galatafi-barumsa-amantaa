Ajajaa lubichi-fuula isaa gara uumataatti deebisee saglee isaa ol kaasuudhaan kan armaan gadii haa jedhu. Uumannis akka seeraatti haa jedhan.
Ha. Lu. Eebbi Waaqayyoo isinii wajjin haa ta’u.
Ha. Uu. Afuura kee wajjin.
Ha. Lu. Uumaa keenyaa haa galateffannu.
Ha. Uu. Dhugumaan ni ta’aaf
Ha. Lu. Yaada laphee keessanii jabeessaa (cimsaa)
Ha. Uu. Waaqayyo bira jirra. Yaa Abbaa keenyaa (3x) yaa Gooftaa gara qorumsaatti nu hin galchin.
Ha. Lu. Ifa kan kennu, Waaqayyo Abbaan, hundumaaf humnaafi jabina kan ta’u, lubbuu hundumaa kan daawatu. Addunyaan osoo hin uumamin dura kan ture, addunyaas kan uume, namootas gara karaa jireenyaatti kan geessu, gammachuu hin darbine mootummaa samii kan kennudha. Gufuu duukanaa keessaa baastee ifa hin argamne kan nuu kennite, nu warra sitti amanne hidhaa gamsiisaa jalaa nu baaste amantiidhaan kan nu kabajje.
Amantoota tajaajiltoota isaarra kan hin fagaanne, addas kan hin baane, yeroo hundumaa isaanii wajjiin kan jiraattu. Sodaadhaann lubbuu kadhattu jalaa kan hin caallisne. Yaada dursee hundumaa kan beeku, kan ta’u otoo hin ta’in dursee kan beeku, kan qoratu, otoo nuti hin kadhatin kan nuti barbaannu beekee kan nuu kennu, otoo hin shakkin kan kadhannu kan nuu dhaga’u. Ifin hin qoratamne, samiirra kan jiran Ergamootarra bulee kan jiraatu, galata anga foonni Ergamootaa dhiyeessan kan simatu. Yaa Waaqayyo kan si kadhannu nu dhaga’i. Jecha callisuun itti hin jirre amantii nuu kenni. Siin jabinaan akka galateeffannuuf, siin akka kabajnuuf sitti amannes jabaannee akka jiraannuuf, yaa Gooftaa nutti maatiin kee tajaajiltootni kee si galateeffanna.
Ha. Uu Yaa Gooftaa siin ni galateeffanna.
Yaa Iyyesuus Kiristoos ejjannoo kadhannaa keenyaa dhaga’i. Warra duudaniif (hin dhageenyeef) sagalee, Warra cabaniif hirkoo, Warra ija dhabaniif ifa, warra naafataniif kan ittin deeman miilla, kan taateef, lamitsii kan qulqulleessitu, dhukkuba hamaadhaan kan qabaman kan fayyiste. Gurra duude kan dhaga’u kan gootu, du’as dheekkamtee duukkana kan saaqxe. Ifa kan uume darbuuf jijjiiramuun kan irra hin jirre. Aduun faanosiin hin banne, qulqulloota irra bulee yeroo hundumaa aduun ibsu, kuta eyyama murtaa’een miidhagina addunyaaf hundumaa kan uume. Namoota fayyisuuf hubatama guddaadhaan hundumatti mullateetta’o. Qoricha ta’udhaan lubbuu kan deebiste, hundumtuu akka ta’utti dursitee beektee uumteetta’o, Ergamoota kan uumte, lafas kan uumte, miidhagina addunyaa yaa Abbaa hundumaafi Gooftaa hundumaa,
addunyaan otoo hin uumamin dura kan ture, ogummaafi beekumsa abba biraa gara biyya lafaatti kan ergame jireenyi kun kan hin jijjiiramne, kan hin diigamne (raafamne) kan hin qoratamneedha. Kan hin mul’ane afuura kana kan dubbatte ati galateefamaadha. Dhugaa ba’umsikees kan ajaa’ibsiisamedha. Kanaaf nuti maatiin keefi tajaajiltoonni kee si galateeffanna. Yaa Waaqayyo si galateeffana.
Ha. Lu. Gartuu (lammii) namootaa keessaa waaqolii akka jadhamaniif, humna seexanaa hundumaa akka moonuuf Afuuraa qulqulluu akka nuu kennite, kan hin hiikamne akka hiiknuuf nama taatee (ta’uu kan hin jijjiiramne sitti amanuuf amantii kan nuu kennite yaa Goofta. Ulfinaafi galata kana yeroo sadaffaafi sii dhi’eessina. Hidhaa du’aa ganuuf amantiidhaan akka injifannuf kan nu taasiste. Lubboota warra sitti amananiif laphee qajeelaa kan uumteef keetiin) Abbaa kee wajjin kan olitti nu araarsite, tokkoos kan nu taasiste, yaa Waaqayyo warra si kadhataniif dhaga’i. Yaa Waaqayyo nutti warri si kadhannu hin himatamin hin midhaminis, yemmuu miidhaminuuf yemmuu himatamnus diinota keenya irra jiraadhu mo’i malee, gatii kadhannaa keenyaa nuu kenni. Goyyomsaa diina keenya seexana irraa akka ofi eegnuuf yaa Mootii hin darbinee kadhannaa keenya nuuf dhaga’i. Maanguddoota nu jabeessi, Ijoollee abbaafi haadhi irraa du’an eebba keetiin simadhu.
Dhiiga cubbuutiin kan goyyoman arjummaa keetiin qulqulleessi. Dadhaboota yaadadhu, beektota taasisi. Gara gantummaatti deemanii kan miidhamaniifi kan badan deebisi, warra hidhaa jiranis fayyisi. Hunduma keenyaafis da’oo ta’i. Yaa Gooftaa Uumaa ulfinaafi mootummaan kun keeti.
Ha. Uu. Ameen.
