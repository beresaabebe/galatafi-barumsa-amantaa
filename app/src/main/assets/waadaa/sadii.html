Ajaja: Lubbichi fuula isaa gara bahaatti deebisee sagalee ol qabuudhaan akka seera Amantii Ortodooksiitti yeedaloon Qulqulluu, haa jedhu.
Uummannis Lubbicha hordofuudhaan yeedaloon tokkummaan haa jedhan (jala haa qaban).

  • Lubni haa jedhu (Ha. Lu) Qulqulluu.
  • Uummanni haa jedhu. (Uu. Ha)

Qulqulluu Waaqayyo, qulqulluu jabaa qulqulluu jiraataa, kan hin duune, qulqulleetii dubroo Maariyaam irra kan dhalatte yaa Gooftaa nu dhiisi.
Qulqulluu Waaqayyo, Qulqulluu jabaa, qulqulluu jiraataa kan hin duune, yordaanositti kan cuubbamee fannoo irratti kan fannifame yaa Gooftaa nu dhiisi.
Qulqulluu Waaqayyo, Qulqulluu jabaa, qulqulluu jiraataa kan hin duune, guyyaa sadafaattis warra du’an irraa adda ba’e kan ka’e galataan gara samiitti ol ni ba’e.
Mirga Abbaa isaa ni ta’e. Dabalees warra jiraniif warra du’an irratti murteessuuf Gooftummaa isaan ni dhufa. Yaa Gooftaa dhiifama nuu taasisi. Galanni Abbaaf haa ta’u, galanni Ilmaaf haa ta’u, galanni Afuuraa qulqulluuf haa ta’u. Har’aas yeroo hundumaas bara baraan ameen. Ameen haa ta’u. Yaa adda kan taate Waaqayyo dhiifama nuu taasisi.
