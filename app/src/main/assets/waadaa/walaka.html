Ha. Lu. Galanni Waaqayyoof haa ta’u.
Ha. Uu. Duguma ni taa’af.
Ha. Lu. Hunduma kan uumte yaa Waaqayyo hin qoratamnee siif qaama keenya ni bichisiifna, yaa Gooftaa galata ganamaas siif ni dhi’eessina. Beekumsa hundaa irraa jabaa kan taatu yaa Waaqayyo, dhiifamni kee kan baayyate, lubbuu kan uumtee Addunyaan otoo hin uumaman dura Abbaa irra kan dhalate, siin ni galateeffanna. Dubbata Abbaa qofa kan taatu, qulqulloota irra bulee kan jiraatu, galata callisuun itti hin jirreen jecha galata raajota Ergamootaan kan galateeffamtu. Angafoota warri qaban raayyaan Ergamootaas kan si galateeffatan.
Harkaan kan hin uumamne uumama ijaan hin mul’anne kan uumte, kan hin qoratamne yaa uumaa qulqulluu galateeffamaa, haa ta’u jatte kan dubbatte, innis iccitii tokkummaafi sadummaa keetii kan ibsu, ogummaa isaa kan nutti hime, ulfina hin qoratamne, isinan dhaalchisa jette abdii nuu taasifte, galata guutuufi qulqulluu siif ni dhiyeessina. Nuti maatiin keef tajaajiltoonni kee si galateeffanna uumannis siin ni galateeffatu.
Ha. Uu. Yaa Gooftaa si galateeffanna.
Ha. Lu. Yaa Waaqayyo ifaa argama jireenyaa, jalqaba beekumsaa, ulfina garamummaa dursuudhan, ulfina surraa, surraa guutudhaan kan nuu kennite. Lubbuu kan uume hayyana qulqulluu kennuudhan kan fayyadu. Argama ogummaa barsiisaa qulqullootaa hundee addunyaa, kadhannaa qulqullootaa kan simattu, kan si dursuuf kan sitti annu kan hin qabne yaa Ilma Abbaa siin ni galateeffanna. Jalqaba angafaa kan taatu yaa dubbata Abbaa nu warra si kadhannuu hundumaaf kan ga’u surraa kee kan nuu laatte mudaan kan irra hin jirre qulqulluu Abbaa adda ta’e, horii daana’oon hin nyaaneef hin balleesine kan qabdu, laphee isaanii qulqulluudhaan kan sitti amananiif kan kennitu, addunyaan otoo hin uumamin kan ture ifa akkeenyaa keetii akka arganiif, Ergamoota kan gammachiisu egduu keenya isa hin geeddaramneedha. Duukkana nura ture eyyama abbaa keetiin kan nurraa fageessite, duukkanaa gara ifaatti kan nu baaste, du’a irraa nu olchitee jireenya kan nuu laatte, cubbuuf (seexanaaf)
bitamuurraas bilisa baastee, fannoo keetiin gara abbaa kee isa samiirra jirutti kan nuu dhiyeessite. Yaa Waaqayyo Wangeelaan karaa jireenyaa nuu qajeelchitee nuu gageessite raajotaanis nu jajjabeesudhaan kan nu dhi’eesite Waaqayyoon sidha. Dabaliis yaa Waaqayyoo beekumsa kee nuu ibsi. Yaa Waaqayyo keenyaa siif galata ni dhiyeessina, galata guddaadhaan nuti maatiin keef tajaajiltootni kee si galateeffanna, uumannis si galateeffatu.
Ha. Uu. Yaa Waaqayyo nu tajaajiltootni keesi galateeffanna.
Ha. Lu. Hundumaa ol kan taate yaa Iyyesuus Kiristoos akkenyaa Ilma Abbaa Gooftummaan kee bara baraaf kan ta’u, mootummaa kee wajjin sagalee keenyaan galata kana yeroo sadaffaaf siif ni dhiyeessina. Uumamtoonni hundinuu sodaaf ollachuudhaan si galateeffatu, lubbuu hundinuu kan sodaatuuf kan Waaqeffatu, lubbuun qulqullootaa sitti amananii siin jabaatanii jiraatu. Qilleensonni sochoosanii kan nutti fidan dhangala’a lolaa hamaa kana nurraa ifatte, badii nu olchitee buufata jireenyaa kan nuuf taate warra galaana irratti dhiphatan kan fayyistu, bakka honaa (gammojii) warra jiraanis kan fayyistu, mana adabaatti hidhaa jabaarra jiran duukaas kan jirtuu, hidhaa du’aa irra kan hiiktu, kan adda nu baaste, kan rakkataniif kan boo’an kan jabeessu, kan gammachiisu, kan dadhabanis fannoo isaatiin kan fayyisu, kan itti amannu nurraa balaa hundumaa kan fageessu, raajottif, duuka bu’oonni dhoksaan kan si galateeffatan yaa Waaqayyo si galateeffanna, siif galata ni dhiyeessina. Siitti amannee mootummaa keeti akka boqonnuuf, fedha (eyyama) kee hojjachaa ajaja keetiin jabaannee akka deemnuuf nu taasisi. Hundumaa arjummaa keetiin daawwadhu, warra xixiqqoof warra gurguddaas, bitaafi eegaa uumataa kan ta’u mootiifi macca isaa eegi.

Yaa Waaqayyo keenyaa ulfaataan mootummaa kan keetihoo addunyaa dursee galatni hin raawatamne Abbaaf Ilmaaf Afuuraa qulqulluuf haa ta’u. har’as yeroo hundumaas dhalootaa dhalootatti hanga bara baraatti kan hin raawatamne.
Ha. Uu Ameen.
