Dugdaa Kirubeel qubbaa kabajaan kan jiraattuu Yaa Waaqayyoo Uumaa keenyaa Ergamoota biratti kabajaan ni galateeffamta galmaa kee Qulqulluutti seennee akka si galateeffannuuf carraa kan nuuf laatte sidha. Qulquluu galata kee yaadadhu galata galgalaas akka siif dhiyeesinuuf kadhannaa hundaa keenyaa nuuf dhag’i. Yeroo iccitii samii irraa amantiidhaan yaanu amantii keenya irra malee, bu’aa bayii malee, moo’amuu nu oolchi, miillota keenya gufuu irraa, ijoota keenya imimmaan dangalaasuu irraa, Qaama keenya du’aa cubbuu isaanii yaaditii garee warra sitti amananii fayyanii nu taasisi. Ilma kee isa tokkicha waliin kabajaaf qalata Afuura qulqulluu wajjin siif haa ta’u. Ameen!

Yaa Gooftaa koo Iyyesuus Kiristoos cubbuun kan guutamte ati guutuu mana kootii jala seenuun naaf hin ta’u. Ani si gaddisiiseera hojii hamaas hojjedheera akka bifaafi fakkeenya keetii kan uumte fooniif lubbuu koo abboommii kee diiguun xureesseera hojii gaarii tokkoo illee hin qabu.

Haa ta’u iyyuu malee, waan na uumteef ana fayyisuuf ijaa nama taateef sababa Qulqulluun fannoo kee fayyisaa kan ta’ee qulqulluu du’a keetii guyyaa sadaffaattis du’a mootee ka’uu keetiin Gooftaa koo yakkaaf abaarsa mara irraa akka na qulqulleesituuf sin kadhada.

Iccitii qulqullina keetiis yeroo fudhadhutti kana itti dheekkamamuuf ceepha’amu akka natti hin taane na fayyisi dhiifama naaf taasisi jireenya addunya kan taate ati yaa Gooftaa dhiifama cubbuu fayyina lubbuu naaf keenni qulqullettii dubroo Maariyaamiif jedhii araarsummaa cuuphicha yoohannisiin kabajamoota kan ta’an Ergamoota, Wareegamtoota kan ta’an waan gaariitiif kan qabsa’aan kadhannaa Qulqulloota maraan hangaa bara baraatti. Ameen!

Gooftaa koo Iyyesuus Kiristoos qulqulluun foon keetiif dhiiga kee dheekkamsa natti hin ta’in fooniif dhiiga koo kan qulqulleessu haa ta’u. Ameen!

Nama fayyisuuf nama kan ta’e iccitii waaqummaa kee kanan fudhadhe (simadhe) anaan galanni kee arraba koo irra, gammachuun kees qaama koo guuti. Iccitii hin himamneen “Qulqulluu, Qulqulluu, Qulqulluu” jedhamtee kan galateeffamtu, sadummaaf tokkummaan kan waaqeefamtu fooniif dhiigi kee osoo natti hin murteesin lubbuuf akka naaf ta’uuf naaf kenni Gooftummaan kee akka ana irratti mul’atuuf firi hojii si gammachiisu naaf kenni fedha kees hojjecha akkan jiraadhu na gargaari.

Si amanuudhaan Abbaa koo jedheen sin waama mootummaa kees nan waama yaa Waaqayyoo maqaan kee nurratti haa galatoomu galateeffamaaf kabajaamaa kan tate yaa gooftaa ati jabaadha siif kabajni bara baraan haa ta’u. Ameen!

Gooftaa fayyisaa kan ta’e Abbaa irraa kan ba’e Afuuraa qulqulluutti ni amannaa Abbaaf Ilmaa wajjin isaaf ni uggumna isas ni galateeffanna innis raajota irra bulee kan dubbateedha.

Hundumaa ol kan taatu duuka bo’oonni kan walitti qaban ishee tokkitti qulqulluu mana Kiristiyaanatti ni amanna cubbuun kan ittiin haqamu cuuphaa ishee tokkittiittis ni amanna jireenya dhufuuf isa bara baraan. Ameen!
