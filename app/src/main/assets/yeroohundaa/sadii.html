Faaruu 129

Yaa Waaqayyo ani qilee keessa taa’een si waammadhe saglee koo naaf dhaga’i.
Gurri kee sagalee kadhannaa kootii kan dhaga’uu haa ta’u.,Yaa waaqayyoo osoo ati cubbuu namaa sakattaatee eenyuutu dhaabbataa?,
Garuu dhiifamni si bira jira. Kanaaf sababa maqaa keef abdiin si gadhadhee,
Lubbuun koo seera keetiin obsite, Sa’ati barii irraa eegalte hanga halkaniitti, Lubbuun koo Waaqayyoon amantee,
Waaqayyoon biraa araara, Isa biraas fayyina heduutu jira,Israa’eel Waaqayyootti haa amanu, Innis Israa’eeliin cubbuu isaa mara irraa isa fayyisa.

Abbaa Ilmaa Afuura Qulqulluu galani haa ta’u bara baraan Ameen!

Hallee luyaa galanni Waaqayyoof hallee luyaa yeroo yerootti galani Amanu’eel waaqa keenyaaf haa ta’u. Ameen!

Yaa waaqa araara gooftaa hundumaa kennaa afuura qulqulluutiin hundumaa kan gammachiiste. Yeroo kanatti akkan si galateeffadhuuf dura kee ijaajjee yeroo eebbifamtuu kanatti kennaa afuura qulqulluu qulqulloota kan ta’an duuka bu’oota irratti akka abiddaa kan irratti roobsiste yeroo kanatti sin galateefadha.

Nama kan jaallattu yaa Waaqa koo ati kadhannaa koo naaf dhaga’i, arjummaa keetiin naaf ergi gufuu fooniif lubbuu irraa na oolchi bara jireenya koo mara waaqeffannaa keetiif bitamee akkan jiraadhuuf ati na qopheessi gara ogummaa afuuraatti yeroo hunda jireenya amalummaa kan ta’uu afuura qulqulluu narra bulchi fedhii foon kootii qofa akkan hordofu nan taasisin.

Hundaa kan bittu abbaa Gooftaa keenya, qoricha keenya, waaqa keenya Iyyesuus Kiristoos arjummaa kee irraa kennaan barbaada.

Nama kan jaallattu Waaqa koo cubbamaa kan ta’u kadhannaa ana garbicha keetii dhaga’i kdhannaan koo dura keetti naaf haa dhiyaatu iyyi koos gara ketti

Yaa waaqa koo gooftaa ani gara kettan abdadhe hamaaniif cubbuun fooniif afuura koo irraa akka fagaatuuf, bultoo afuura qulqulluu akkan ta’uuf araraaf arjummaa kee akkan hawwuf obsaaf kasooma (of-gadii qabuu) naaf kenni. Qormaata natti dhufu mara obsaaf galataan akkan simadhuuf bara jireenya koo mara ilmi kee Iyyesuus Kiristoos gidiraa inni fannoo irratti fudhate yaaduun akkan baadhu na taasisi. Eebba kee qulqulluu sana qaama kootti aguugi dhiifamaaf araara dura keetti akkan argadhuuf Iyyesuus Kiristoos haboo Waaqummaatiin qaama namummaatiin faaruu ergmootaatiin marfamee murtiif sa’atii dhufutti warreen itti murteefamuu wajjin akkani hin lakka’amnee arjummaan kee naaf haa kennamu.
Siif Ilma kee Kiristoosiif amalummaa jireenya kee afuura qulqulluuf kabajni galanni bara hanga baraatti hata’u. Ameen!
